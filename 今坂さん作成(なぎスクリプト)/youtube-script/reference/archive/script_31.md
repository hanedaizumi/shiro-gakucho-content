# 第2章：ラテラルシンキング＝「思考のワープ」

ここで、一つ重要な概念を紹介します。
それが**「ラテラルシンキング（水平思考）」**です。

これだけ聞くと、「なんか難しそう…」と思うかもしれませんが、安心してください。
中学生でもわかるように説明します。

思考には、大きく分けて2つの種類があります。

一つは、**「ロジカルシンキング（垂直思考）」**。
これは、地面に穴を深く掘っていくような作業です。
「AだからB、BだからC」と、論理を積み上げて正解を導き出す。
階段を一段ずつ登っていくイメージですね。
学校教育やビジネスの現場で求められるのは、主にこっちです。

でも、この思考法には弱点があります。
それは、「掘った穴の先に岩盤があったら終わり」ということです。
「階段が壊れていたら進めない」ということです。
真面目な人ほど、この壁にぶつかった時に、「もっと強く掘らなきゃ！」「スコップが折れたら手で掘ればいいじゃない！」と、**もはや工事現場の鬼監督みたいになって、血眼で穴を掘り続けてしまう。**

「いやいや、それは極端でしょ」と思いましたか？
でも、あなたも無意識にやっているんですよ。
「残業すればなんとかなる」「土日も働けば終わる」
これ、まさに「手で穴を掘っている」状態です。

そこで登場するのが、もう一つの**「ラテラルシンキング（水平思考）」**です。
これは、穴を深く掘るのではなく、**「別の場所に穴を掘る」**作業です。
あるいは、階段を登るのではなく、**「エレベーターに乗る」**とか、**「隣のビルから飛び移る」**といった発想です。

一言で言えば、**「チート（裏技）」**です。
「真面目に戦うのをやめて、ルールそのものを疑うこと」です。

## 「正解」を探すな、「別解」を作れ

ここで、ラテラルシンキングの威力を示す有名な話を紹介しましょう。
ある古いビルで、「エレベーターが遅い」というクレームが殺到しました。

**ロジカルシンキング（垂直思考）**の解決策はこうです。
*   「最新の高速エレベーターに入れ替える」（数千万円かかる）
*   「制御プログラムを最適化する」（専門家が必要）

どれも正攻法ですが、コストも時間もかかります。
しかし、**ラテラルシンキング（水平思考）**ができる人は、たった数万円でこのクレームをゼロにしました。
何をしたと思いますか？

答えは、**「エレベーターホールの壁に鏡を設置した」**です。

鏡を置いたことで、待っている人は身だしなみを整えたり、自分の姿を見たりすることに集中し、「待たされている」というイライラを感じなくなったのです。
問題は「エレベーターの速度」ではなく、「待っている時間の退屈さ」だったと再定義したわけです。

「え、そんなの屁理屈じゃん」と思いましたか？
「問題は解決してないじゃん」と。

でも、クレームはゼロになったんです。
結果が出れば、それが正解なんです。

これが「思考のワープ」です。
真正面から壁を壊すのではなく、視点をずらして問題を消滅させる。

多くの人が悩んでいるのは、目の前の壁を「どうやって乗り越えるか」ばかり考えているからです。
「壁が高い」「壁が硬い」と嘆いている。
でも、ラテラルシンキングができれば、「壁の横を通り抜ける」とか「壁にドアを作る」、あるいは「鏡を置いて壁を忘れさせる」といった、誰も思いつかない解決策が見えてきます。

真面目な人ほど、「壁は乗り越えるものだ」という固定観念に縛られています。
でも、目的は「壁を越えること」ではなく、「向こう側に行くこと」ですよね？
なら、手段なんてなんだっていいんです。泥臭くても、ズルくても、向こう側に着けば勝ちなのです。
