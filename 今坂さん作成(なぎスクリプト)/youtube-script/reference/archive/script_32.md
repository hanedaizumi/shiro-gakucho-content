1.  **究極のゴール（遠くの星）**:
    死ぬまでに達成したい、ワクワクする大きな夢。

「お金は手段であって、目的じゃない」と。

# 第3章：行動（Action）

そして、人間の意志力（ウィルパワー）は有限です。
朝起きて「今日何しようかな…」と考えた時点で、あなたの脳のエネルギーは浪費されています。
だから、**「考える前に動く仕組み」**を作るんです。

*   **朝**: 起きたらすぐに白湯を飲む。5分だけ散歩する。
*   **夜**: 明日やることを１つだけメモして寝る。

これだけです。
「え、そんなことでいいの？」と思いましたか？
いいんです。

「決めたことを守れた」という事実。
これが自己肯定感を積み上げます。
「俺、ちゃんとやってるじゃん」
この小さな自信が、やがて大きな挑戦へのエネルギーになります。

いきなり「毎朝1時間勉強する」なんて無謀なルーティンは組まないでください。
三日坊主で終わって、余計に自信を失うだけです。
まずは「水を飲む」。ここから始めてください。
朝起きた瞬間、格闘技を身につけたかと思うぐらい体が軽いはずです。

## ルール6：成長の記録をつける（可視化）

*   理想の未来に1mmでも近づけたか？

これを書くだけです。
「今日は本を1ページ読んだ」
「今日は怒らずに子供の話を聞けた」
そんな些細なことで構いません。

これを記録することで、脳は
「お！ こいつは成長しようとしてるな！」
と認識し、勝手に成功するための情報を集め始めます（RAS機能）。

RAS（網様体賦活系）とは、脳のフィルター機能のことです。
普段、私たちは街中で「赤いポスト」を意識しませんが、「ポストを探そう」と思った瞬間に、街中のポストが目に飛び込んできますよね？
これと同じです。
「成長」を意識して記録し始めると、脳は日常の中から「成長の種」や「成功のチャンス」を勝手に探し始めるんです。

日記なんて続かない？

騙されたと思って、今日から「できたこと」を1つだけメモしてみてください。
