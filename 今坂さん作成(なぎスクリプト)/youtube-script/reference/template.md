## テンプレート運用メモ（構成の流れ）

| 段階 | 役割 | 含めること |
|------|------|------------|
| **フック** | 3秒で離脱させない | ハイライト、共感する一言、結論の先出し |
| **導入** | 信頼と期待を作る | 挨拶、自己紹介、悩みの言語化、目次、特典予告 |
| **本題① なぜ** | 納得と共感 | 2つの壁／原因、共感ストーリー、禅・科学の根拠 |
| **本題② どのように** | 再現可能にする | 2〜4ステップ、実演・例文、ツール名と受け取り方 |
| **まとめ** | 記憶に残す | ステップ要約、感情メッセージ、行動の一言 |
| **CTA** | 次の行動へ | 特典の価値、受け取り方、LINE・メルマガ誘導 |
| **エンディング** | チャンネル継続視聴 | 登録・高評価の匂わせ、感謝、また次回 |

## 視聴者維持のテクニック（ト書きで入れておく演出）

- **（ハイライト）** … 冒頭に配置。離脱防止用の一番の見せ場を先出し
- **（テロップ：〇〇）** … キーワード・章名・感情を一言で
- **（映像：〇〇）** … BGMの止め方、静寂、切り替え、実演の指示
- **（※実際に〇〇を表示）** … 画面共有・スクショ・AI画面など
- **（※ここで目次を表示）** … 目次のタイミング
- **（※場面転換）** … フックから本編への切り替え
- **問いかけ** … 「〜ですよね？」「どうでしょう？」を適宜挿入
- **共感パート** … 「私もそうでした」「分かります。痛いほど」＋過去の失敗談
- **番号・リスト** … ステップ1・2・3、TOP5、目次の明示で「どこまで見たか」を分かりやすく
- **特典の先出し** … 「動画の最後で〇〇を渡します」で最後まで見る理由を作る
