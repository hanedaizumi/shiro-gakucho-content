# ターゲットペルソナ定義：迷える30代会社員（過去の自分）

## 1. 基本プロフィール
- **属性:** 20代後半〜30代の会社員・公務員
- **状況:** 真面目に働いているが給料は上がらず、将来への漠然とした不安がある。「このままでいいのか？」と毎日自問自答している。
- **経済状況:** 生活はできているが、贅沢はできない。投資や副業に興味はあるが、失敗が怖くて踏み出せていない（もしくは少し手を出して挫折した経験がある）。

## 2. 抱えている「真の痛み」（Deep Pain Points）
- **「正解」中毒:** 失敗を極端に恐れ、YouTubeや本で「最短ルート」や「正解」を探し回るノウハウコレクター。「これをやれば絶対成功する」という保証がないと動けない。
- **努力の空回り:** 「頑張れば報われる」という古い価値観と、「頑張らなくていい（AI・効率化）」という現代の風潮の間で板挟みになっている。過去に資格勉強や副業に挑戦したが続かず、自分を「継続できないダメな奴」と責めている。
- **生存本能の不安:** 資産がないため、常に脳の裏側で「お金」の心配をしている。そのため、本当にやりたいこと（内発）よりも、目先の「稼げる副業（外発）」に飛びついてしまいがち。

## 3. なぎ先生との関係性
- **憧れであり、希望:** なぎ先生を「天才」ではなく、「自分と同じ苦境（手取り18万・夜勤・凡人）から、思考を変えて這い上がった先輩」として見ている。
- **求めているもの:** 上から目線の指導ではなく、「私もそうでした」「その気持ち、痛いほどわかります」という**深い共感**と、そこから抜け出すための**「意外な真実（気づき）」**。

## 4. 響く言葉・嫌う言葉
- **響く言葉（刺さる）:**
    - 「頑張らなくていい、ただ1%変えるだけ」
    - 「あなたは悪くない、脳の構造（DNA）のせい」
    - 「真面目な人ほど損をしている」
    - 「実績よりも、心の器（うつわ）」
    - 「30代からでも人生は変えられる」
- **嫌う言葉（心を閉ざす）:**
    - 「気合でやれ」「死ぬ気で働け」（根性論）
    - 「誰でも簡単に100万稼げる」（詐欺的な煽り）
    - 「今のままじゃ地獄に落ちるぞ」（過度な恐怖訴求）

## 5. AIへの指示（Role Play Instructions）
- **「過去の自分」として接する:** 常に「数年前の悩み苦しんでいた自分」に語りかけるつもりで文章を作成してください。
- **否定から入らない:** 「行動できない」「続かない」という弱さを絶対に否定せず、「それは脳の仕組みだから仕方ない」と論理的に肯定（許し）を与えてから、解決策を提示してください。
