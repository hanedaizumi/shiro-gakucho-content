# なぎ先生の「弱み」エピソード集
（台本作成時、ファン化のための「弱み（Weakness）」として引用可能なデータです）

1. **ゲームでの大人気なさ**: 家族でゲームをする際、子供相手に本気を出し、負けると「もう一回！」と勝つまでやめずに妻に呆れられる。
2. **外出前の指差し確認**: 外出前は必ず妻から「トイレ行った？」「スマホ持った？」と指差し確認をされている。
3. **プリンへの執着**: 普段は「執着を手放す」と語っているのに、冷蔵庫に入れておいたプリンを家族に食べられただけで半日以上根に持つ。
4. **巨大フィギュア**: お金の賢い使い方を発信している裏で、巨大なフィギュアを買って妻に呆れられる。
5. **口論での連敗**: 論理的思考をしているはずなのに、妻との口論には全く勝てない。
6. **靴下の裏返し**: デスク周りの整理整頓は完璧なのに、靴下を裏返しのまま脱ぎ捨てて、妻に叱られる。
7. **家電オンチ**: AIツールは使えるのに、洗濯機やテレビなど、身近な家電の操作が壊滅的に分からず、妻に呆れられる。
8. **ヨレヨレの部屋着**: 「執着を手放す」思想を大切にしているのに、学生時代から着ているヨレヨレの部屋着だけはどうしても手放せない。
9. **詰め込み旅行**: 家族旅行で分刻みの詰め込みすぎたスケジュールを立ててしまい、妻から叱られる。
10. **美容室でのチキン**: YouTubeで自分の意見を発信しているのに、美容室で「もう少し短くしてください」と言う勇気はない。
11. **グミの誘惑**: ストイックな食生活をしているけど、グミの誘惑には勝てない（市場調査、という名目で、新作が出たらすぐに買ってしまう）
12. **おもちゃの独占**: 子供の教育にいいからといっておもちゃやゲームを買うが、自分が一番ハマって妻に呆れられる。
13. **家事の承認欲求**: あんまり承認欲求はないんだけど、家事をやった時は、「どう？」と言わんばかりに妻に褒めてもらいたがる。
14. **仕事への逃避**: 図星を突かれたり、面倒な用事を頼まれたりすると、急に仕事の画面に没頭するフリをして逃げようとしてしまう。
15. **鉄棒への熱中**: 公園で、子供に鉄棒を教えていたら、自分がやるのに夢中になってしまい、早く帰ろうと言われても「待って！」と答える。
16. **衝動買い**: 欲しいガジェットや本を見つけると、後先考えずにすぐポチってしまい、妻から冷たい目で見られる。
17. **アクセサリーの盲点**: 他人の強みは見抜けるけれど、妻がいつもと違うアクセサリーをつけていることは気づけない。
18. **牛乳の買い忘れ**: 常に思考を整えてタスクをこなしているが、妻に頼まれた牛乳を買ってくるのを忘れて呆れられる。
19. **賞味期限の過信**: 自己管理の重要性を発信しているが、賞味期限の食材を「大丈夫でしょ！」といって食べてお腹を壊す。
20. **プッチンプリン至上主義**: 旅をして、色々なものを食べてきたが、結局一番美味しいと思うものはプッチンプリン。
21. **お風呂の遅刻**: 仕事の時間管理は完璧だが、子供とのゲームに夢中になって妻から「早くお風呂入りなさい」と怒られる。
22. **買い物中の迷子**: 買い物に行くと興味を惹かれたものに立ち止まってしまうため、迷子として子供達に捜索される。
23. **家具の謎部品**: 毎日のタスクは過不足なくこなすが、IKEAなどの家具を組み立てると、必ず謎の部品が余る。
24. **指相撲の完敗**: スポーツ万能で身体も鍛えているが、指相撲だけは異常に弱くて小学生の息子に完敗する。
25. **電車での乗り過ごし**: 電車でスマホを使って仕事をすると、集中しすぎて毎回駅を乗り過ごしてしまう。
26. **流行歌の無知**: 最新のAIツールは詳しいが、最近流行りの歌を全く知らずに子供に教えてもらう。
27. **砂糖と塩の混同**: 一緒に仕事をしている人たちへの指示は間違えないが、よく砂糖と塩を間違えてしまう。
