# ポストの3つのパターン

ユーザーからの指示に応じて、以下のいずれかのパターンで出力すること。

* **パターン1：短文ツイート（30～180文字）**
    * 目的：タイムラインで一瞬で目を引き、強烈な気づきを与える。
    * 特徴：装飾を削ぎ落としたパンチライン。1ポストにつき1メッセージのみ。
* **パターン2：長文ツイート（400文字以上）**
    * 目的：論理とストーリーで深く納得させ、プロフ遷移やコミュニティ参加へ繋げる。
    * 特徴：エピソードから入り、残酷な現実を提示し、論理的な解決策（Web3/トレード）へ導く。
* **パターン3：マインド発信ツイート（30～300文字）**
    * 目的：投資家・事業家としての基準値を示し、フォロワーの思考を一段引き上げる。
    * 特徴：ノウハウではなく「習慣」「哲学」「思考法」に焦点を当てる。

# 行動指針（X特化ルール）

1.  **形容詞の排除（事実で殴る）**：「すごい」「簡単」「便利」は禁止。数値、名詞、具体的な行動に置き換える。
2.  **一次情報の注入**：一般論は書かない。必ず `episode.md` 等から実体験・失敗・検証結果を引き出し、シロ学長の独自理論として展開する。
3.  **極限のターゲット特定**：30代〜50代男性の中で、「これは自分のことだ」と刺さる1人を脳内に設定し、その人が目を逸らしている「残酷な現実」を突きつける。
4.  **説教臭さ・AI感の排除**：感情的な共感やAI特有の「〜ですよね」「重要です」は不要。断定口調で、ユーザーの思考を一段下から持ち上げる構成にする。
5.  **抽象化→転用（絶対ルール）**：トレース元のフレーズ・単語・具体的事例は一切そのまま使わない。「骨格（構造）だけ借りて、肉（内容）は全部シロ学長のものに置き換える」こと。読んだ人がトレース元と見比べた時に「パクリだ」と気づくレベルの転写は厳禁。
6.  **情報補完ルール**：インプット情報が足りない場合は、①トレース元アカウントの別ポストを参照、または②Webで関連情報を検索して補完すること。補完した情報源はポストのメモ欄に記載する。

# ワークフロー（厳守事項）

『★Xトレース元ポスト.md』に格納されているポストを元に、以下のStep1〜Step4を順番に実行すること。

### Step 1. 構造のトレース（抽象化）
トレース元ポストの「なぜこのポストは読まれるのか？（フックの強さ、心理的トリガー、論理展開）」を分析し、要素を抽象化する。いきなり文章は書かず、思考の骨組みだけを構築する。

### Step 2. 3案の作成（全投稿タイプで必須）

**朝ポスト（@8btc3スタイル）は以下の構造を必ずトレース：**
1. 謎めいた断言1文フック
2. 年号の数列でサイクルを視覚化（例：「2015 → 2019 → 2023 → 2026」）
3. 「これまで一度も例外なく〜同じ形を繰り返してきた。」
4. 「そして今、〇年。"あの形"に近づいている。」
5. シロ学長の実績数字（野村證券・過去の価格予測等）で信頼性担保
6. 「もう一つ、今かなり危険なシグナルが出ている。」
7. 見かけと真実の逆転（「一見〜に見えるが、この相場では違う」）
8. 「振り落とされるか。それとも仕掛ける側に回るか。」
9. **「いいねとブックマークして爆益を取る準備をして下さい」**（必須CTA）

**夜ポスト（@FxRumasanスタイル）は以下の構造を必ずトレース（1,000字以上の長文記事）：**
1. 名言フック（「〜なのではなく〜だけです。」構文）
2. 経験年数＋「正直に言います。」で権威と謙虚さを同時表現
3. 「例えば、」＋ ■ 箇条書き（過去の典型的NG行動）
4. 失敗の具体ストーリー（数字・情景を使って）
5. 「その頃の私に伝えたいこと」で内省に転換
6. NG感情を正当化（「これは人として自然な反応」）→ 「ただ、相場には１つだけ残酷な性質があります。」で反転
7. ✅ 社会の美徳がトレードでは致命傷になるリスト
8. 変化のきっかけ（「勝ち方探しをやめた」）
9. ✅ やめた行動リスト
10. ・シンプルな習慣リスト
11. 「地味。退屈。格好良くない。でも〜」
12. トレードの本質を語る大結論
13. **「リプか引用リポストで感想をお待ちしています」＋次記事ティーザー**（必須CTA）

**昼ポスト（@fxdragonking88スタイル）：**
- 100〜150字で短く即時性高く。具体的な水準と方向性を提示。3案出しは不要。

### Step 3. 最優秀案の選定（AIが選定・理由を必ず明記）
作成した3案の中から、以下の基準で最優秀案を1案選定する：
- ターゲット（30-50代男性）への訴求力が最も高いか
- シロ学長のゴール（トレード・仮想通貨への誘導）に最も直結するか
- トレース元の構造を最も正確に踏まえているか
- シロ学長固有の一次情報が最も活かされているか

**選定後はユーザーへの確認なしに即Step 4へ進む。**

### Step 4. ブラッシュアップ（最終仕上げ）
* 冒頭フックの強化（前提を否定できているか？）
* 反論の事前処理（「怪しい」「時間がない」などの疑念を潰せているか？）
* スマホでの可読性（改行や空白の調整）
* 一次情報の注入（episode.md・trade_principles.mdのエピソード・数字が入っているか確認）
を実施し、完成版として出力する。

### Step 5. 出力
`08_アウトプット/Xポスト/x_YYYYMMDD.md` に**ブラッシュアップ済みの最終版のみ**を保存する。  
3投稿（朝・昼・夜）を1ファイルにまとめ、各ポストの直後に画像取得方法を記載する。  
3案と選定理由はCursor会話上で提示し、出力ファイルには含めない。
