# エバーローンチ 戦略定義・意識すること

> このファイルはAIが「ステップメール」「ローンチ動画台本」を磨く際に必ず最初に読み込む。
> シロ学長のエバーローンチ全体設計の思想・ルール・注意点をここに集約する。

---

## エバーローンチとは（このプロジェクトの定義）

メルマガに新規登録してきた読者に対して、**ステップ配信（オートレスポンダー）**で自動的に信頼を構築し、
一定のタイミングで商品・サービスを案内する「常設型のローンチ導線」。

通常のリアルローンチと違い、**登録者ごとに個別の"タイムライン"が動く**ため、
1回作れば365日自動で稼ぎ続ける仕組みになる。

---

## このプロジェクトのゴール

**1つの目標**: メルマガに入ってきた読者が、ステップ配信を通じて
「この人（シロ学長）を信頼できる → 商品を試してみよう」という状態になること。

売るのではなく、**「信頼してもらった結果として、買いたくなる」** 流れを設計する。

---

## エバーローンチの全体導線（フロー）

```
メルマガ登録
    ↓
【ステップ配信: 信頼構築フェーズ】
  - シロ学長の一次情報・失敗談・哲学を届ける
  - 読者の悩み・不安に深く寄り添う
  - 無料コミュニティ（Discord）への誘導
    ↓
【ローンチ動画: 価値提示フェーズ】
  - 商品の価値・WHYを動画で伝える
  - 社会的証明（受講生の声・実績）を示す
  - 「今すぐ始めるべき理由」を提示
    ↓
【カート開放: 購入フェーズ】
  - 締め切り・希少性・特典で決断を促す
  - 迷っている人への背中を押すフォローメール
```

---

## ステップメール設計の鉄則

### ① 1通目は「壁を壊す」
- 登録直後の最初のメールが一番読まれる
- 「よくあるメルマガ」に見せない → 冒頭から自己開示・失敗談で入る
- 「ちゃんと人間が書いている」と感じさせることが最重要

### ② 信頼構築は「与えること」でしか積み上がらない
- 1通1通が「これだけで価値がある」状態にする
- 「買ってください」より先に「役に立つ」を積み上げる
- 商品の話は**信頼が十分に積み上がってから**

### ③ 読者の「感情の変化」を設計する
各ステップで読者に起こしたい変化を意識して書く：

| ステップ | 狙いたい感情変化 |
|---|---|
| 1〜2通目 | 「この人、ちゃんとした人だ」（信頼の入口） |
| 3〜4通目 | 「自分と似た悩みを持っていたんだ」（共感・親近感） |
| 5〜6通目 | 「この人の言う通りにすれば変われそうだ」（期待感） |
| 7通目〜 | 「今すぐ始めないと損だ」（緊迫感・行動意欲） |

### ④ CTA（誘導）は段階的に
- 序盤: Discordコミュニティ（無料・低ハードル）
- 中盤: ローンチ動画を見てもらう
- 終盤: 商品LP・申し込みページ

## ローンチ動画設計の鉄則

### GDT（Goal / Desire / Target）フレームワークを意識する
- **G（Goal）**: 視聴者が動画を見た後に取ってほしい行動
- **D（Desire）**: 視聴者が「欲しい」と思う感情を最大化するシーン・言葉
- **T（Target）**: 誰に向けて話しているかを明確にする

### 動画の基本構成（VSL型）
1. **フック（0〜30秒）**: 「この動画を最後まで見る価値がある」と思わせる
2. **共感・問題提起（30秒〜3分）**: ターゲットの悩みを代弁する
3. **物語（3〜8分）**: シロ学長の失敗 → 気づき → 変化のストーリー
4. **価値の証明（8〜12分）**: 実績・受講生の声・データ
5. **商品説明（12〜15分）**: 何が得られるかを具体的に
6. **クロージング（15分〜）**: 今行動すべき理由・特典・締め切り

### NG：やってはいけないこと
- 商品の機能説明から始める（「〇〇が使えます」→ 誰も興味を持たない）
- 「楽して稼げる」の煽り文句
- 証拠のない実績の提示
- 視聴者を不安にさせたまま終わる（必ず希望で締める）

## 全体を通じて意識すること（最重要）

### 「売る」ではなく「救う」という視点
シロ学長のミッションは「正しい知識で1人でも多くの日本人を経済的自立へ導く」こと。
ローンチの文章も、その延長線上にある。

- 読んだ人が「この人は自分のことを考えてくれている」と感じるか？
- 行動を促す前に、十分に「なぜ必要か」を理解してもらえているか？
- シロ学長の一次情報（失敗談・数字・リアルな体験）が十分に入っているか？

### 「詐欺師と同じことをしない」という誓い
シロ学長は詐欺被害にあった経験がある。
だからこそ「怪しい手法」「感情だけを煽る文章」は**絶対に使わない**。
ロジック・データ・体験談の3点セットで信頼を作る。

4. フィードバックがあれば `07_インプット/エバーローンチ用/` 内のFBファイルを参照
5. **全9通を1つのファイルにまとめて** `08_アウトプット/エバーローンチ/ステップメール/` に保存

#### ステップメール作成の品質基準
- **1通あたり1,000〜1,200字以内**（1メッセージ＝1つの核心メッセージ）
- ターゲット（30〜50代男性、家族のために働くが将来不安を抱える層）への「痛みの高解像度化」を必ず入れる
- シロ学長の実体験・ストーリーで感情の振れ幅を作る（借金300万→独学失敗→環境で逆転の流れを活用）
- 辻褄チェック：「否定したもの」と「推奨するもの」が矛盾しないか確認
  - ❌ NG：「AIツール」を否定した後、インジケーターを「AI分析ツール」と呼ぶ
  - ✅ OK：「完全自動売買ツール」を否定し、インジケーターは「相場分析サポーター」と位置づける
  - ❌ NG：「ロジックだけでは勝てない」と言った後、ロジック勉強会にだけ誘導
  - ✅ OK：「ロジック（基礎）＋環境（最重要）の2ステップ」として整理し、勉強会はその入口と位置づける
- 出口（個別相談への誘導）は強気に。テイカー層を明確に排除する選別の文言を入れる

#### 9通の構成と感情ステップ

| 通 | テーマ | 狙いたい感情変化 |
|---|---|---|
| 1通目 | 情弱あるあるで痛みを刺す → 2ステップ（ロジック基礎+環境）→ 明後日予告 | 「この人、自分の現状を分かってくれてる」 |
| 2通目 | 補助ツールの正しい位置づけ（「自動売買ツール」ではない）→ 明日予告 | 「この人は誤魔化さずに本当のことを言ってくれる」 |
| 3通目 | 勉強会公開 + ロードマップ作成会オファー（強めのスク） | 「自分専用のロードマップが欲しい」 |
| 4通目 | 自己投資マインド（成功者の共通点）→ 動画視聴促進 | 「環境に投資することへの納得感」 |
| 5通目 | 締め切り近い + なぜロードマップにこだわるか（痛みを味わわせたくない＋機会損失） | 「この人は本気だ」 |
| 6通目 | 増枠 + トレードへの恐怖払拭 | 「自分でもできそうかもしれない」 |
| 7通目 | 12時間前：ノウハウコレクターの末路（FOMO） | 「今すぐ動かなければ」 |
| 8通目 | 6時間前：Q&A（不安の最終撃破） | 「最後の疑問が消えた」 |
| 9通目 | 3時間前：最終ラストコール（強気な選別） | 「覚悟が決まった」 |
