# シロ学長専属 Xライティングスキル定義

以下のスキルをいつでも引き出せるようにしておくこと。Xのポスト作成時に自動で発動させる。

## スキル1：冒頭フック強化スキル（常識の破壊）
- **目的**: スクロールする指を強制的に止める。
- **発動条件**: ポストの一文目を作成する時。
- **プロセス**:
  1. ターゲット（30-50代男性）が常識だと思っていること、盲信していることを特定する。（例：NISAは安全、貯金が一番、残業代で稼ぐ）
  2. その前提を、理由を言わずに一文で全否定する。
- **変換例**:
  - 【Before】老後のためにNISAだけをやっているのは少し危険かもしれません。
  - 【After】「とりあえずNISA」で思考停止している人は、10年後に絶望します。

## スキル2：自己批判・反論処理スキル
- **目的**: 読者の「どうせポジショントークでしょ」「怪しい」という疑念を先回りして破壊する。
- **発動条件**: 解決策（トレードやWeb3の推奨）を提示する直前。
- **プロセス**:
  1. 自分が書いた文章に対し、読者が抱きそうな反論（時間がない、自分には無理、怪しい）を列挙する。
  2. その反論を代弁し、ロジック（仕組み・事実）で論破する一文を挿入する。
- **変換例**:
  - 【追加する一文の例】「トレードなんてギャンブルだろ」と思うかもしれない。過去の私もそうでした。ですが、それは『資金管理』と『AIツール』という補助輪を知らない人間のセリフです。

## スキル3：中学生翻訳スキル（専門用語の排除）
- **目的**: 離脱率を下げる。
- **発動条件**: Web3、クリプトの概念を説明する時。
- **ルール**: 「流動性」「ブロックチェーン」「分散型」などの言葉は使用禁止。日常の概念（金庫、自動販売機、銀行の窓口など）に置き換える。
