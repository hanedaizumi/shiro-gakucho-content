## CursorからナノバナナをMCPで直接使う方法

ナノバナナはGoogleのGemini 2.5 Flash Imageをベースにしたサービスです。
Cursorから直接呼び出す方法は、以下の2段階です。

### STEP 1：Google AI Studio APIキーを取得する

1. [Google AI Studio](https://aistudio.google.com/) にアクセス
2. 「Get API Key」からAPIキーを発行（無料枠あり）
3. このAPIキーでGemini 2.0 Flash の画像生成機能を呼び出せます

> ナノバナナのサービスそのものはWebUIのため直接API連携はできませんが、同じモデル（Gemini 2.5 Flash Image）を直接APIで呼び出せます。

### STEP 2：CursorのMCPに登録する

Cursorの設定ファイル（`~/.cursor/mcp.json`）に以下を追加：

```json
{
  "mcpServers": {
    "image-gen": {
      "command": "npx",
      "args": ["-y", "@google/generative-ai-image-mcp"],
      "env": {
        "GOOGLE_API_KEY": "あなたのAPIキー"
      }
    }
  }
}
```

> ただし、現時点でGoogle公式のimage-gen MCPパッケージはベータ段階のため、
> **最も確実な方法は「Cursorのチャットで画像生成を依頼する」** ことです。
> CursorにはすでにAI画像生成機能が内蔵されており、チャット内で「〇〇のイメージ画像を作って」と依頼するだけで画像生成できます。
> ナノバナナのWeb UIは https://trynanobananaai.com/ja から無料で使えます。
