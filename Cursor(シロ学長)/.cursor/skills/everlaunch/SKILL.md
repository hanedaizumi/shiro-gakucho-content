# エバーローンチ コンテンツスキル

## 事前読み込み（必須）

- `shiro-master` スキルを読み込み、ゴールと思考の軸を確認する
- `goal-seek` スキルの「5. エバーローンチ」セクションを確認する
- `07_インプット/エバーローンチ用/` のインプットファイルを確認する
- `08_アウトプット/エバーローンチ/` の過去コンテンツを確認する

---

## エバーローンチの全体構造

```
① 認知（YouTube・X・メルマガ → 新規流入）
        ↓
② 興味（ステップメール：シロ学長を知る・信頼する）
        ↓
③ 参加（無料セミナー/Webinar：価値を体験する）
        ↓
④ 検討（セミナー後フォローメール：背中を押す）
        ↓
⑤ 成約（スクール・VIP参加）
        ↓
⑥ 定着（Discord・継続コンテンツ）
```

---

## コンテンツ種別

| 種別 | 説明 | 格納先 |
|------|------|--------|
| ステップメール | 登録〜セミナー参加までの自動メール | `08_アウトプット/エバーローンチ/ステップメール/` |
| セミナー台本 | ウェビナー・セミナーの台本 | `08_アウトプット/エバーローンチ/セミナー台本/` |
| ローンチ動画台本 | 購入促進のための動画台本 | `08_アウトプット/エバーローンチ/ローンチ動画/` |
| LP（ランディングページ） | 申込・購入ページのコピー | `08_アウトプット/エバーローンチ/LP用/` |

---

## 作成ルール

1. **常に「視聴者の問題」から始める** → シロ学長の実績・自慢から始めない
2. **問題提起 → 共感 → 解決策 → 証拠 → CTA の順番を守る**
3. **ハードルを段階的に下げる** → 無料セミナー参加 → 体験 → 購入の順
4. **強制感を排除する** → 「登録しないでください（本気の人だけ）」の逆説型を活用
5. shiro-master の「コンテンツ作成の型（4ステップ）」に準拠する
