# ゴールシークスキル（タスク別ゴール定義）

> 「何を作るか」の前に「何のために作るか」を確認する。
> このスキルは shiro-master と必ずセットで使う。

---

## ゴールの階層構造（全タスク共通）

シロ学長のコンテンツには3層のゴールがある。

```
最終ゴール（ビジネスゴール）
  └ 中間ゴール（各タスクのKPI）
       └ 即時ゴール（このコンテンツで達成すること）
```

**最終ゴール（常に意識すること）：**
1. Discordコミュニティへの入会
2. メルマガ登録
3. セミナー・スクール参加 → 成約

**全コンテンツの役割：** 上記3つのいずれかへ視聴者・読者を自然に導くこと。「ただの情報発信」で終わらせない。

---

## タスク別ゴール定義

### 1. YouTube企画（企画立案・サムネ・タイトル）

**最終ゴール：** 再生数を増やし、新規視聴者をチャンネルに引き込む

**中間ゴール（KPI）：** 拡散率 2〜5倍以上（リップル系は2倍以上、BTC系は5倍以上）

**即時ゴール（このタスクで達成すること）：**
- サムネを見た瞬間に「自分ごと」と感じさせる
- タイトルを読んだだけで「続きが見たい」と思わせる

**成功の判断基準（このコンテンツはOKか？）：**
- [ ] ターゲット（30〜50代男性）の痛みや欲求に直結しているか
- [ ] 数字・固有名詞・逆説のいずれかが含まれているか
- [ ] 「知らなかったら損をする」という緊急性があるか
- [ ] サムネとタイトルが同じ方向を向いているか（ちぐはぐでないか）
- [ ] 過去の動画と企画コンセプトが被っていないか（`03_アーカイブ/01_YouTube_台本アーカイブ/`で確認）

---

### 2. YouTube台本

**最終ゴール：** 視聴維持率を高め、チャンネル登録・Discord参加・メルマガ登録を増やす

**中間ゴール（KPI）：** 視聴完了率、チャンネル登録率、コメント数

**即時ゴール（この台本で達成すること）：**
- 最後まで見た視聴者が「この人は本物だ」と感じる
- 「今日から何をすればいいか」が明確に伝わる

**成功の判断基準（この台本はOKか？）：**
- [ ] 「この人俺の知らないことを知っている！」という瞬間が1箇所以上あるか（情報でファン化）
- [ ] 「可能性」「かもしれない」を使わず、根拠つきで断言しているか（断言力）
- [ ] 専門用語はすべて中学生でも分かる例えに翻訳されているか（わかりやすさ）
- [ ] 前半・中盤・後半にCTAが自然な流れで配置されているか
- [ ] エモい話（自己開示）が「売り込み」ではなく「共感」として機能しているか

**台本のゴール設計チェック（開始前に確認）：**
```
この台本を最後まで見た視聴者に、どんな感情・行動変化を起こさせたいか？
→ 感情：______________（例：「この人すごい、信頼できる」）
→ 行動：______________（例：「Discordに登録してみよう」）
→ 知識：______________（例：「ダウ理論の本当の使い方が分かった」）
```
この3点を書いてから台本作成に入ること。

---

### 3. Discord VIP配信

**最終ゴール：** VIPメンバーの継続率を上げ、口コミ・紹介を生む

**中間ゴール（KPI）：** VIPメンバーの熱量・活動頻度・継続期間

**即時ゴール（この配信で達成すること）：**
- 「VIPコミュニティに入っていて良かった」という体験を作る
- 「これはYouTubeでは聞けない情報だ」という特別感を届ける

**成功の判断基準（この配信はOKか？）：**
- [ ] 「ここだけ感」（YouTubeやXでは出さない情報）があるか
- [ ] シロ学長自身の投資行動・本音が開示されているか
- [ ] 根拠（数字・過去事例）が含まれているか
- [ ] 「VIPコミュニティにいるからこそ得られた価値」が体験できるか

---

### 4. メルマガ

**最終ゴール：** 読者を「信者化」し、セミナー・スクール参加につなげる

**中間ゴール（KPI）：** 開封率、クリック率、セミナー参加率

**即時ゴール（このメルマガで達成すること）：**
- 読み始めて30秒で「自分ごと」と感じさせる
- 読み終えた後に「次のメールも絶対読もう」と思わせる

**成功の判断基準（このメルマガはOKか？）：**
- [ ] 冒頭の1〜2文で読者の「今の状況・感情」を的確に言い当てているか
- [ ] シロ学長のエピソードが「自慢」でなく「共感」として機能しているか
- [ ] メールの目的（何をしてほしいか）が1つだけ明確に示されているか
- [ ] 「今日読んでよかった」という感覚を与える情報・気づきが含まれているか
- [ ] CTAは自然な流れの中に置かれているか（突然の売り込みになっていないか）

**メルマガの構成型（毎回この型で書く）：**
```
① 読者の今の状況に共感する1〜2文
② シロ学長の自己開示（似た経験・失敗談）
③ 今日届けたい「気づき・知識」
④ 行動促進（今日から1つだけ）
⑤ 締め（次回への期待か、CTAへの自然な誘導）
```

---

### 5. エバーローンチ

**最終ゴール：** セミナー参加 → 体験 → 購入の自動化された流れを作る

**中間ゴール（KPI）：** セミナー参加率、成約率

**即時ゴール（このコンテンツで達成すること）：**
- 「このセミナーを受けないと損をする」という感覚を作る
- 「自分でもできそう」という確信を持たせる

**成功の判断基準（このコンテンツはOKか？）：**
- [ ] 問題提起 → 共感 → 解決策 → 社会的証明 → CTA の5段階が揃っているか
- [ ] 「視聴者/読者の問題」から始まっているか（シロ学長の実績から始まっていないか）
- [ ] セミナー参加のハードルが十分に下がっているか（無料・短時間・気軽に参加）
- [ ] 購入後のビフォーアフターが具体的に示されているか

**エバーローンチのゴール設計チェック（開始前に確認）：**
```
このコンテンツを受け取ったターゲットに、どんな状態変化を起こさせたいか？
→ Before（今の状態）：______________
→ After（目指す状態）：______________
→ 行動：______________（セミナー参加・購入 など）
```

---

### 6. Xポスト

**最終ゴール：** フォロワーを増やし、YouTube・Discord・メルマガへ送客する

**中間ゴール（KPI）：** インプレッション・エンゲージメント率・プロフィールクリック率

**即時ゴール（このポストで達成すること）：**
- スクロールを止める（1行目で引き留める）
- 保存・リポストされる（「後で使いたい」と思わせる）

**成功の判断基準（このポストはOKか？）：**
- [ ] 1行目が断言・逆説・数字のいずれかで始まっているか
- [ ] 1スクロール以内で読み終えられるか（長くなりすぎていないか）
- [ ] 「保存したい」「シェアしたい」と思える具体的な知識・気づきがあるか
- [ ] シロ学長らしいトーン（地獄を知る先輩・断言力）が出ているか

## ゴール確認プロセス（作業開始前に必ず実行）

どのタスクであっても、以下の3問に答えてから作業を開始すること：

```
Q1. このコンテンツを受け取った人に、どんな「感情変化」を起こしたいか？
    → ________________

Q2. このコンテンツを受け取った人に、最終的にどんな「行動」をとってほしいか？
    → ________________（チャンネル登録・Discord参加・メルマガ登録・購入）

Q3. このコンテンツを通じて届ける「1つの核心メッセージ」は何か？
    → ________________（これが軸。台本・文章を通じてブレない）
```

この3問への答えが決まったら、shiro-masterの「コンテンツ作成の型（4ステップ）」に沿って構成する。
