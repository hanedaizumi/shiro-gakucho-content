# VIP限定配信作成スキル（Discord向け）

## ベーススキルの読み込み（最初に実行）

1. `shiro-master` スキルを読み込む → エピソード選択・断言力の軸を確認
2. `goal-seek` スキルの「3. Discord VIP配信」セクションを確認 → VIPメンバーの熱量を上げることを意識する

## コンテンツの定義

VIP限定配信とは、シロ学長が**自分の投資行動・本音・限定情報**をVIPメンバーだけに届けるコンテンツ。
YouTubeやXには出さない「ここだけ感」と「自己開示」が最大の価値。

### 主なコンテンツパターン

| タイプ | 内容例 | 使用シーン |
|--------|-------|-----------|
| **投資行動開示型** | 「今日〇〇を買い増しした」「ポジションを切った」 | 自分が動いたタイミング |
| **相場深掘り分析型** | 「数年に一度の異常事態が起きている」「この指標が〇〇を示している」 | 通常の市況解説より深い内容 |
| **マインド型** | 「こういう局面でどう考えるか」「暴落で動ける人と動けない人の差」 | 相場が荒れている局面 |
| **成果共有型** | 「今日のトレードでこうなった」「メンバーがこう動いた」 | 成果が出たタイミング |

## 文章作成ルール

### 必須要素
1. **「ここだけ感」を最初に出す** → タイトルや冒頭で「VIPだから言える」を明示
2. **自己開示から入る** → 「今日〇〇しました」「自分は〇〇と見ています」を先に言う
3. **根拠を数字・固有名詞で示す** → 「なんとなく怖い」ではなく「〇〇という事実があるから」
4. **過去の類似事例と結果を引用** → 「同じことが2022年に起きた。その後どうなったか」
5. **VIPへのメッセージで締める** → 「VIPメンバーだからこそ使い倒してほしい」

### 禁止事項
- 「楽して稼げる」「絶対上がる」などの断言（法律リスク・信頼失墜）
- 「不安な方はこちら」的な煽り ← 不安を煽った後に**必ず根拠と希望をセット**にすること
- 過度に長い → 1,000字以内が目安。深い内容ほどテンポよく切る

### トーン
- 「地獄を知る先輩が、信頼する仲間に本音で話す」感覚
- 上から目線ではなく、**同じ相場を戦っている仲間として**語りかける
- ただし確信がある時は断言する（「〇〇だから、後悔しない」）

## 出力フォーマット

```
# [タイトル（ここだけ感のある一言）]

## [冒頭の自己開示]
（今日/今週/先ほど 何をしたか・どう見ているか）

---

## [なぜ今なのか？根拠の説明]
（数字・固有名詞・過去事例を使って）

---

## [結論・確信の表明]
（〇〇まで落ちても後悔しない／だから今動く、など）

---

## [VIPメンバーへのメッセージ]
（この環境を使い倒してほしい、という締め）
```
