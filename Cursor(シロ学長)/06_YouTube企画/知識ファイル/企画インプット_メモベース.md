# 企画インプット メモベース（知識ファイル）

> 出典: 企画インプット(メモベース)
> バズる動画・サムネイルのヒントとなる観察メモ集。企画立案時の「勝ちパターン設計図」として活用すること。

---

## 観察メモ（バズる要素のパターン）

### ビジュアル・デザイン系

- **整ってるものは目に留まる / 丁寧なくらし** → 「整理整頓」「洗練されたライフスタイル」系は刺さる
- **ぐちゃぐちゃから整うのいい** → ビフォーアフター構造（散乱→スッキリ）は強い
- **同じものが集合していると目を引く** → 同種のものを並べる構図は視認性が高い
- **手伸びてるのおもろい** → 動きのある手・ジェスチャーの画像は目を引く
- **トトロいいな** → 親しみのあるキャラクター・温かみのあるイラスト
- **繊細なタッチ→アニメだけどリアル** → リアルとアニメの中間（半リアル）は新鮮味がある
- **ヒビ入ってるのおもろい** → 「壊れる」「崩れる」ビジュアルは意外性がある
- **富士山は日本の象徴 / 芭蕉の顔が趣ある** → 日本的シンボルは親しみと権威性を同時に演出

### タイトル・コンセプト系

- **S&P500とオルカン否定→富裕層は買わない** → 常識の否定・逆張りは強い（T要素）
- **世界一やさしい〇〇** → G要素（楽して学べる）に訴える
- **2026先取り良いかも** → 時事・未来予測は希少性（T要素）を刺激
- **お金に困らない富のマインドセット** → D要素（富への欲望）に直接訴える
- **第3の時間☆** → 意外性のある切り口（「第3」という表現が気になる）
- **仕事ができる人がキリ悪い時間にやっていること** → 「それな！」という強い共感とノウハウ感
- **インド人は悩まない / 孤独に生きよ→逆張り** → 逆張りタイトルは意外性（T要素）が強い
- **男ともだち（どっちなん？に答えを出す）** → 疑問を提示して答えを引き出す構造

### バズワード・フォーマット系

- **だまされない / 1000万円超 / お金を守る** → 「守り」文脈も刺さる（恐怖回避のD要素）
- **2050年問題** → 社会的課題・将来不安（T要素の希少性・危機感）
- **永久保存版 〇〇 大全** → 「大全」「完全版」系バズワードの可能性あり
- **すぐに使える / いっきに学び直す** → G要素（時短・即効性）への訴求
- **まとめて整理する → だるそうな物をシンプル化** → 煩雑なものを「スッキリ」整理するコンセプト
- **エレガントな毒の吐き方 → やってはダメなことを肯定した上で方法をレクチャー** → 禁止事項を肯定する逆転の構造（T要素）
- **着るだけで疲れがとれる服** → 「するだけで〇〇できる」という簡単・受動的訴求（G要素）
- **哲学は人生の道しるべ** → 権威性と深みを演出するコンテンツ

### 視覚的フック（右脳）

- **先が見えるかどうか？ゴールが見えるか？が重要** → サムネイルで「ゴールのビジュアル」を見せる
- **ゴルフスイングしてる感じの右脳強い** → 動作・スポーツ系の躍動感あるビジュアル
- **ロレックス図録** → 高級品・憧れのビジュアルはD要素（富への欲望）を強く刺激
- **変な地図** → 「え？これは何？」という視覚的な意外性（T要素）
- **トランプがもたらす** → 時事の著名人を使ったサムネイルは注目を集める

### シロ学長への応用メモ

- 現在バズっているワード：**「リップル」「金持ち確定」** → 急変させず維持すること
- 逆張り（「NISAだけでは不十分」「銀行預金が最大のリスク」）系は強い
- 「整理整頓」「スッキリ」構造を仮想通貨・トレードの複雑さに当てはめると刺さる可能性あり
- 「富のマインドセット」「サイドFIREの日常」系のライフスタイル訴求も有効

## 【シロ学長チャンネル】拡散率2倍以上の動画サムネ 共通項分析

> 2026年5月時点、シロ学長チャンネルで拡散率が通常の2倍以上だった動画のサムネを分析した結果。
> 企画立案・サムネ設計の際は必ずこのパターンを参照すること。

### 共通項①：マスコットキャラクター（シロ学長クマ）が必ず登場

- シロ学長のクマキャラクターが全高拡散サムネに共通して登場
- **ブランド認知×親しみやすさ** の両立。「シロ学長のチャンネルだ」と一瞬で分かる
- 著名人（トランプ・イーロン）との組み合わせでも、キャラが前面に出ることでシロ学長のフィルター（信頼）が加わる
- → **企画立案時はキャラクターをどこに配置するか必ず設計に含める**

#### キャラクター詳細（シロ学長マスコット）

- 外見：白いホッキョクグマ、黒の学士帽（金タッセル付き）、丸眼鏡、タブレットを持っている
- 印象：「地頭の良い先生が教えてくれる」「怪しくない・信頼できる」という安心感
- チャンネルカラー：白×黒×金（サムネの配色と統一）

#### キャラクター 7つの配置パターン（右脳訴求に必ず適用）

**パターン①：驚き・指差し配置**
- 表情：目を見開き、口が開いた衝撃顔。チャート・数字・ニュースを指差す
- サイズ：画面の30〜40%（存在感を出す）
- 配置位置：左端または右端
- 用途：T要素（意外性）の強い企画。「これを見て！」「え、こんなことが？」
- 例：「アメリカ承認でリップル急騰？」→ APPROVEDスタンプを見て驚き指差し

**パターン②：確信・OKサイン配置**
- 表情：自信満々の笑顔。親指を立てる、またはうなずいている
- サイズ：画面の20〜30%
- 配置位置：右端（テキストの隣で「これが正解」という承認）
- 用途：「確定」「確実」系の断言型企画。元野村の権威で情報を担保する
- 例：「200円→10,000円確定」→ 確信顔でOKサイン

**パターン③：警告・ストップ配置**
- 表情：真剣な表情で手のひらを前に出す（ストップジェスチャー）
- サイズ：画面の35〜45%（存在感が必要）
- 配置位置：中央左
- 用途：「売るな」「するな」「待て」系の行動制止型企画
- 例：「リップル売るな」→ 手のひらを前に出してストップ

**パターン④：タブレット教授配置（デフォルト構図）**
- 表情：真剣またはやや驚き顔。タブレットを視聴者の方向に向けて見せている
- サイズ：画面の25〜35%
- 配置位置：左端（「私が教えます」という構図）
- 用途：「〇〇の理由」「元野村が解説」「根拠を言います」系の教育型企画
- 特記：キャラクターがすでにタブレットを持っているため、タブレット画面に「チャート」「グラフ」「データ」を表示する合成が自然に成立する

**パターン⑤：驚喜・ガッツポーズ配置**
- 表情：嬉しそうにガッツポーズ、または飛び跳ねている
- サイズ：画面の30〜40%
- 配置位置：左端または中央
- 用途：「金持ち確定」「〇〇万円確定」「FIRE確定」系の利得訴求型企画
- 効果：「視聴者の未来の感情」を先取りして視覚化する

**パターン⑥：覗き見・秘密配置**
- 表情：何かを隠れて見ているような表情。目だけ出している、または横から覗いている
- サイズ：画面の20〜25%（小さめで「隠れている」感を演出）
- 配置位置：左下または右下のコーナー
- 用途：「隠す」「漏らした」「裏技」「バレた」系の反社会性T要素企画
- 例：「銀行が隠す暴騰シナリオ」→ 銀行の陰から覗き見ているキャラ

**パターン⑦：小キャラ・ブランディング配置**
- 表情：通常の笑顔
- サイズ：画面の10〜15%（小さいブランド要素として）
- 配置位置：左上または右上のコーナー
- 用途：著名人（トランプ・CEO等）が主役の場合にブランド認知として添える
- 効果：「このシロ学長のチャンネルの動画だ」という既存視聴者への認知シグナル

#### キャラクター配置の3原則

1. **サムネの感情テンション（驚き・喜び・警告）とキャラの表情を必ず揃える**
   - テキストが「売るな（警告）」なのにキャラが笑顔では情報が矛盾する

2. **キャラのタブレット画面は「補足データ」の表示スペースとして活用できる**
   - チャート・数字・グラフを小さくタブレット内に表示すると情報密度が上がる

3. **著名人・リアル写真が主役の場合はキャラを小さく（パターン⑦）、著名人がいない場合はキャラを大きく（パターン①〜⑤）**

### 共通項②：「小さい元手→大きなリターン」の変換構造が明示

- 0.1BTCで → 2400万円確定
- 20万円買えば → 金持ち確定
- リップル400枚で → 金持ち確定
- **左：今の自分が持てる金額 × 右：到達できる未来の金額** という対比構造
- → 「今の自分でもできる」という心理的障壁の低さと「これだけ増える」というD要素の両立
- → **企画立案時は必ず「元手（入口）→リターン（出口）」のペアで数字を設計する**

### 共通項③：「確定」ワードの戦略的使用

- 「金持ち確定」「2400万円確定」「FIRE確定」が高頻度で登場
- 「可能性がある」「かもしれない」ではなく**断言形**が拡散を生む
- ただし動画内で根拠を回収できる場合のみ使用すること（サムネ詐欺防止）
- → **根拠が動画内で示せる数字×シナリオにのみ「確定」を使用する**

### 共通項④：著名人・権威機関の活用

- トランプ、イーロン・マスクなど世界的著名人の写真が高頻度で登場
- 「〇〇が漏らした」「〇〇が保証」という主語設定で権威性を最大化
- 著名人＝「この情報は本物だ」という信頼の代理証明になる
- → **企画立案時は「誰が言った・保証した情報か」という権威の出所を必ず設計する**

### 共通項⑤：「漏らした」「隠す」「裏技」系の反社会性ワード

- 「トランプが漏らした」「資産10兆円投資家が漏らした」「税金ゼロの裏技」
- **T要素（反社会性）**：「本来は知られるべきでない情報が明かされている」という構造
- 視聴者に「自分は特別な情報を手に入れた」という優越感を生む
- → **企画立案時に「誰かが意図的に隠していたこと」という構造を入れられないか検討する**

### 共通項⑥：シンプルな2〜3色配色＋数字が最大フォント

- 黒×金×白、または青×白×金が主流
- 全サムネで**金額の数字が最も大きいフォントサイズ**になっている
- スマホ画面（小サイズ）でも金額が一瞬で読める視認性を確保
- → **右脳訴求の画像より金額テキストが最前面に来るレイアウト設計を優先する**

### 共通項⑦：「ゴール（未来の自分）」を直接見せる

- FIREの豊かな生活、金持ちのライフスタイルをサムネで直接表現
- 「この動画を見れば自分もこうなれる」というゴールビジュアルが右脳を直接刺激
- → **サムネイルには「投資した後の未来の自分の状態」を可能な限り視覚化する**

### 共通項⑧：「〇〇ない人の悲惨な未来」型の恐怖回避訴求

- 「0.01BTCない人の悲惨な未来」という恐怖回避（D要素）
- 「自分は該当するかも」という不安でクリックを生む
- → **利得訴求（金持ち確定）と損失回避訴求（知らないと損）を交互に使い分ける**

### 高拡散サムネ 設計チェックリスト（企画立案時に確認）

- [ ] マスコットキャラクター（クマ）の配置を設計したか？
- [ ] 「元手（入口）→リターン（出口）」の数字ペアを設定したか？
- [ ] 「確定」を使う場合、動画内で根拠が回収できるか？
- [ ] 権威者・著名人・機関の「誰が言った」を設計したか？
- [ ] 「漏らした」「隠す」「裏技」などT要素の反社会性を入れられるか？
- [ ] 金額数字が最大フォントになるレイアウトか？
- [ ] 「ゴール（未来の自分）」または「しない場合の損失」をビジュアルで表現できるか？
