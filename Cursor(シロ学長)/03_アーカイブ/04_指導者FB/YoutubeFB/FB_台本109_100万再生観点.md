# YouTube台本改善レポート：台本109
## リップル(XRP) 1000円到達動画

**FB日：2026/6/9**
**元タイトル：【99%が知らない】2026年中にリップル(XRP)が1000円に到達する衝撃の理由【仮想通貨】**

---

## 全体評価

この台本は構成力が高く、ペルソナ設定も明確。ただし、100万再生を狙うレベルにするには以下の観点で改善が必要。

1. 冒頭2秒のフックが弱い（ショート的思考の欠如）
2. 未回収ワードの設置が甘い
3. 既視感のある表現が多く差別化が弱い
4. 感情コントロール（BGM設計含む）の指示が不足
5. LINE誘導の位置とタイミングが最適でない

---

## 1. タイトル改善

**改善前**
```
【99%が知らない】2026年中にリップル(XRP)が1000円に到達する衝撃の理由【仮想通貨】
```

```
リップル500万溶かした男が断言する、2026年XRP1000円の3つの根拠【仮想通貨】
```

**理由**
- 「99%が知らない」は仮想通貨系で使い古されたフックであり、既視感が強すぎてクリック率を下げる

- 「3つの根拠」で具体性を担保
- 「衝撃の理由」は抽象的でベネフィットが不明確。視聴者は「衝撃」を求めているのではなく「自分が儲かるかどうか」を知りたい
- SEOキーワード「リップル」「XRP」「2026」「1000円」「仮想通貨」はすべて維持

## 3. 「この3つを知れば〜」パートの改善

**改善前**
```
この３つを知れば、取引所のアプリを開いた時に見える含み損の赤い数字を見ても、
心臓がバクバクしなくなります。
逆に、この3つを知らないままだと、目先の値動きに振り回されて投げ売り。
まだ持っていない人は、リップルの可能性に気づかないまま2027年を迎え、
大きな機会損失をすることになります。
これ、めちゃくちゃもったいなくないですか？
買うチャンスは全員に与えられているのに、
知ってるか？知らないか？で1年後の資産が軽く5倍は変わる。
最後まで見れば、次暴落が来た時に、
震えることなく、笑顔で買い増しボタンを押せるようになるので、
集中して聞いていってください！
```

**改善後**
```
この3つを知ったあと、あなたは取引所のアプリを開くのが怖くなくなります。
含み損の赤い数字を見ても、「ああ、想定通りだ」と思えるようになる。
逆に知らないままだと、次の暴落で耐えきれず投げ売り。
そして売った3日後に爆上がりして、枕を殴ることになります。
まだ持っていない人はもっと危険です。
2027年の正月に「あの時買っていれば…」と後悔する側に回る。
この動画は15分です。
たった15分で、1年後の資産が5倍変わる可能性がある。
集中して、最後まで聞いてください。
```

**理由**
- 「もったいなくないですか？」は問いかけが弱い。視聴者に考えさせる余裕を与えると離脱リスクが上がる。代わりに具体的なシーン描写（「枕を殴る」「2027年の正月に後悔」）で感情を動かす
- 「15分」と具体的な時間を提示することで、視聴者が投資する時間のコストを認識し、「15分なら見てもいいか」と判断できる。これはコンバージョンの心理学でいうコミットメントの明示化
- 「笑顔で買い増しボタンを押せる」は表現が軽すぎる。仮想通貨で損をしている視聴者にとって「笑顔」は現実味がなく、共感が薄れる

## 5. 本編①「なぜリップルは値上がりしないのか？」の改善

**改善前**
```
リップルの正体は、世界中に一瞬で価値を移動できるインフラです。
って言われてもピンとこないですよね？
実は今、世界中の銀行が海外にお金を送るとき、
「SWIFT」っていう50年前の仕組みを使ってます。
これ、たとえるなら「伝書鳩」です。
```

**改善後**
```
リップルの正体を一言でいうと、
「世界中にお金を3秒で届ける新幹線」です。
今、世界の銀行が海外送金に使っているのはSWIFTという仕組み。
これ、50年前に作られたもので、たとえるなら伝書鳩です。
鳩がメモを運ぶだけ。お金そのものは動いていない。
だから海外送金に3日かかるし、手数料も数千円取られる。
リップルが入ると、この鳩が不要になる。
お金が3秒、手数料は数円で届く。
伝書鳩から新幹線への乗り換え。これがリップルの正体です。
```

**理由**
- 「って言われてもピンとこないですよね？」は自分で自分の説明を否定している。視聴者は「ピンとこない説明をされた」と感じ、信頼性が下がる
- 改善後は最初に結論（新幹線）を出してからSWIFTの説明に入る。視聴者は結論が先にわかっている方が説明を理解しやすい
- 比喩を一本化する。「伝書鳩→新幹線」の対比を最初と最後で挟むことで、記憶に残りやすい構成にする

- 改善後は「聞いたことはありますか？」と問いかけることで、知っている人も知らない人も対応できる。これは複数ペルソナに同時に語りかける技術
- 最後に「そのクジラたちが今、何をしているか？」で次のセンテンスへの橋渡しを作り、視聴継続を促す

## 7. 本編②クジラパート・数値の見せ方改善

**改善前**

**改善後**

- 「偶然だと思いますか？」の一文で視聴者に考えさせ、次の説明への期待値を上げる。これは修辞的疑問文と呼ばれるテクニックで、視聴維持率を上げる効果がある

## 11. まとめパートの改善

**改善前**
```
今日の話を30秒でまとめます。
1.リップルは「1枚の値段」ではなく「時価総額」で見る。
今200円なのは、まだ評価される前段階だから。

3.買って放置だと時間とお金の両方を失う可能性あり。ガチホ×トレードの二刀流戦略が正解。
```

**改善後**
```
最後に、今日の話を3行でまとめます。
1つ目。リップルが200円なのは「安い」のではなく「まだ評価されていない」だけ。
時価総額で見れば、すでにトップクラスの通貨です。
2つ目。2026年、クラリティ法案・クジラの買い占め・アジアの実需、
```

- 最後の2行で「行動しないリスク」を提示し、LINE誘導への自然な橋渡しにする。元の台本のまとめはただの要約で終わっており、次のアクションへの推進力がない

## 12. エンディングの改善

**改善前**
```
私はトレードで失敗して一晩で300万円を溶かした日の翌朝、
コンビニでおにぎりを手に取って、棚に戻したことがあります。
手取りは15万円、1日300円の飯で、食いつなぐ日々でした。
```

**改善後**
```
最後に、1つだけ個人的な話をさせてください。
私が一晩で300万を溶かした翌朝、
コンビニでおにぎりを手に取って、棚に戻しました。
あの時の自分に言えることがあるとすれば、
「感情で売るな。ルールを作れ。そして学べ」
この3つだけです。
今、含み損を抱えて苦しんでいるあなたに、

```

**理由**
- おにぎりエピソードは冒頭に移動したため、ここでは繰り返さない。ただし、別の角度（「過去の自分に言えること」）で再利用することで、冒頭との円環構造が生まれ、動画全体に一貫性が出る
- 「だからこの動画を作りました」で動画の存在意義を語ることで、視聴者との感情的なつながりが生まれ、チャンネル登録・コメントへの動機が自然に生まれる
- 元の台本は「手取り15万円、1日300円の飯」と情報を詰め込みすぎている。感情を動かすには、情報量を減らして余韻を残す方が効果的

```
今動画を見て「よし、わかった」って思っても、
3日後には9割の人が忘れて元の日常に逆戻りします。
なのでそれを防ぐために、メルマガを用意しました。
```

**改善後**
```
ここまで見てくれたあなたに、1つだけ聞かせてください。
今日学んだこと、3日後に覚えていますか？
正直、9割の人は忘れます。
忘れた人は、次の暴落で同じ過ちを繰り返す。
だから、忘れないための仕組みを作りました。
```

**理由**
- 「お願いがあります」は売り込み感が強い。「聞かせてください」の方が対話的で、視聴者の心理的抵抗が下がる
- 「3日後に覚えていますか？」と直接問いかけることで、視聴者自身が「確かに忘れるかも」と内省する。元の台本は「こうなりますよ」と一方的に宣告しているだけ
- 「忘れないための仕組みを作りました」の方が、「メルマガを用意しました」よりベネフィット訴求が明確

## 14. 特典紹介の改善

**改善前**
```
①世界最先端 トレードロジック完全解説動画
→これ1本で、負けないトレード手法が学べます。
②24時間何でも質問し放題！シロ学長のAIチャットボット
→深夜3時に疑問が浮かんでもすぐ聞ける
③資産を何倍にも増やす！Web3必須ツールとサイト集
→私が実際に毎日使っているものだけを厳選
④Web3.0・仮想通貨・DAO・NFT関連用語集
→ニュースが自分で読めるようになる
⑤埋めるだけで完成！ライフプランシート
→「あと何年で2000万円に届くか」が一目で見える
```

**改善後**
```
今なら5つの特典を無料でお渡ししています。
特に重要なのはこの2つ。
1つ目、私のトレードロジック完全解説動画。
今日話した「感情を排除するルール」の具体的な設定方法を、
画面共有しながら全部お見せしています。
2つ目、売買タイミング自動表示ツール。
「いつ買えばいいかわからない」を解決する補助輪です。
残り3つの特典は概要欄に書いてあるので、そちらで確認してください。
登録は無料。合わなければ即解除できます。
リスクはゼロなので、とりあえず受け取っておいてください。
```

**理由**
- 5つの特典を全部口頭で説明すると冗長になり、離脱ポイントになる。相対的な視聴維持率データでは、商品説明が長いシーンで平均を大きく下回る傾向がある
- 「特に重要なのはこの2つ」と絞ることで、視聴者の情報処理負荷を下げる
- 動画本編の内容（感情排除・補助輪）と特典を紐づけることで、「動画の続き」として自然に受け取れる構造にする。元の台本は特典が動画内容と接続されていない

## 15. コメント誘導の改善

**改善前**
```
ちなみに、あなたは今、リップルを持っていますか？
持っている方は『持ってる』 まだの方は『これから買う』
とコメントしてください。
あと、質問もガンガン投げてください。
```

**改善後**
```
最後に、コメントで教えてください。
あなたが今一番不安に感じていることは何ですか？
「含み損が怖い」「いつ買えばいいかわからない」「トレードが不安」
なんでもOKです。
全部読んで、次の動画のテーマにします。
もちろん「リップル持ってる」「これから買う」だけでも嬉しいです。
高評価とチャンネル登録もお願いします。
それでは次の動画でお会いしましょう。
```

**理由**
- 「持ってる/これから買う」の二択は情報量が少なく、コメント欄が単調になる。アルゴリズムはコメントの量だけでなく、文字数や多様性も見ている
- 「一番不安に感じていること」を聞くことで、長文コメントが増え、エンゲージメント指標が向上する
- 「次の動画のテーマにします」と宣言することで、視聴者に「自分のコメントが次の動画に影響する」という参加意識を持たせ、コメント率が上がる

## 構成全体の改善サマリー

| セクション | 改善前の課題 | 改善のキーポイント |
|---|---|---|
| タイトル | 「99%が知らない」の既視感 | 実体験×権威性の掛け算 |
| 冒頭1分 | フックが弱い、自己紹介が早い | 損失体験で開始、未回収ワード3つ設置 |
| ベネフィット提示 | 抽象的で弱い | 具体的シーン描写と時間提示 |
| LINE誘導（前半） | 唐突で露骨 | 正直さで信頼獲得、短く切る |
| 本編説明 | 自己否定的な前置きが多い | 結論→説明の順序で信頼維持 |
| 中間LINE誘導 | 長すぎる | 3行以内に圧縮 |
| 落とし穴パート | 一般論で差別化なし | 具体的相場エピソードで自分ごと化 |
| まとめ | ただの要約 | 行動への分岐点を提示 |
| 特典紹介 | 5つ全部列挙で冗長 | 2つに絞り、本編と接続 |
| コメント誘導 | 二択で浅い | 自由記述で長文コメント誘導 |
