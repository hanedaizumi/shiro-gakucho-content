## 全体所感

台本の骨格は非常に強い。ストーリー構成（導入→前提知識→本題→落とし穴→解決策→CTA）がしっかりしており、比喩の使い方も分かりやすい。改善の方向性は以下3点に集約される。

1. **冒頭の破壊力強化**（1行目のフックが弱い）
2. **サムネのZ構造最適化**（情報過多・好奇心要素が弱い）
3. **CTA配置の調整**（回数が多すぎる・タイミングが問題）

## 1. タイトルのFB

### 改善元
```
【99%が知らない】リップルCEOが絶対に言わない仕込みタイミング5選｜知っている人だけが勝ち続ける
```

### 問題点
- 文字数が約50文字でオーバー（30〜35文字が目安）。YouTube上で後半が切れて表示されない
- 「｜知っている人だけが勝ち続ける」は表示されない上に、ベネフィットとして弱い（抽象的）
- 「99%が知らない」と「絶対に言わない」でフックが二重になり、どちらも薄まっている
- SEOキーワード「リップル」「XRP」は入っているが、「仮想通貨」が抜けている

### 改善案（5パターン）
```
① リップルCEOが絶対に言わない仕込みタイミング5選【2026年最新】
② 【知らないと全損】リップル絶好の仕込みタイミング5選｜XRP
③ リップル99%が知らない仕込み時5選｜買い場はここだけ
④ 【XRP】リップルCEOが隠す仕込みタイミング5選｜資産10倍の分岐点
⑤ リップルいつ買う？CEOが言わない仕込みタイミング5選【完全保存版】
```

### 推奨：②または④

**②の理由**
- 「知らないと全損」→ 恐怖回避のフックが強い（金融ジャンルでは損失回避心理が最強のトリガー）
- 30文字以内でスマホ表示でも切れない
- SEOキーワード「リップル」「XRP」両方カバー

**④の理由**
- 「資産10倍の分岐点」→ 具体的数字＋好奇心
- 「隠す」→ 「言わない」より陰謀感が強くクリック率UP
- 35文字でギリギリ収まる

## 2. サムネイルのFB

### 現状の問題点

| 問題 | 詳細 |
|------|------|
| Z構造の左上が弱い | 「99%が知らない」は使い古されており、フックとしてのインパクトが足りない。左上で止められない |
| 情報量が多すぎる | 3要素が文字で詰まっており、一瞬で理解できない |
| ベネフィットが不在 | 「これを知ると何が得られるか」が視覚的に伝わらない |
| 好奇心要素が弱い | 隠された情報感・ピー隠しなどの「続きを見たい」仕掛けがない |
| 数字の具体性が活かしきれていない | 「5選」はあるが、金額や倍率といった欲望を刺激する数字がない |
| 人物画像の活用が中途半端 | CEOの画像があるなら大胆に使うか、演者の表情を主役にすべき |

### 改善案（5パターン）

#### パターン①：恐怖回避×具体数字型【最推奨】

| 位置 | 要素 | テキスト/ビジュアル |
|------|------|---------------------|
| 左上 | フック（最重要） | 「知らないと全損」（赤文字・最大サイズ） |
| 右上 | 具体 | 「資産10倍の差」（黄文字） |
| 左下 | ベネフィット | 「仕込み時5つ」（白文字） |
| 右下 | 好奇心 | リップルCEOの画像＋口に「×」マーク |

背景：暗めのチャート画像（下落→上昇の矢印）

**推奨理由**
- 左上「知らないと全損」は損失回避バイアスを最大限に刺激し、金融ジャンルで最も止まりやすい
- 「資産10倍の差」は具体的数字で右脳を刺激
- CEOの口に「×」マークで「何かを隠している」感が好奇心を生む
- 要素が4つに絞られており、一瞬で理解できる

### 【導入部分】

#### ① 冒頭1行目

| | 内容 |
|---|---|
| **改善前** | `ハッキリ言って今の時代「リップルが1万円になる！」みたいな情報を追いかけているうちは、あなたの資産は1円も増えません。` |
| **問題点** | 冒頭0.5秒の一言目が勝負なのに「ハッキリ言って」という前置きで弱まっている。「1円も増えません」もベネフィットの裏返しとして弱い |
| **改善後** | `「リップルが1万円になる！」──こういう情報を追いかけてる人、全員負けます。` |
| **理由** | 冒頭一言目に「リップルが1万円になる！」というキャッチーなセリフを持ってくることで、視聴者が瞬時に「自分のことだ」と感じる。「全員負けます」は断定的で離脱を防ぐフックになる |

## 4. 全体構造に対するFB

### 良い点（そのまま維持すべき）
- 3部構成（前提→本題→落とし穴＋解決策）が明快で視聴者が迷わない
- 比喩が豊富で金融初心者にも分かりやすい（新幹線・山の地図・フレンチ料理）
- バフェットの引用→「実は違う解釈」という構成が視聴者の知的好奇心を刺激
- 主婦の成功事例がペルソナ拡大に貢献している
- ガチホ→二刀流のフレームワークに独自性がある

### 全体で改善すべき点

| 項目 | 問題 | 改善方向 |
|------|------|----------|

## 5. 100万再生を狙うための追加アドバイス

### ① ハイライト（冒頭アバン）の追加【必須】

現状の台本にはハイライト（本編のダイジェストを冒頭に持ってくる構成）がない。100万再生を狙うなら、冒頭0〜15秒に本編の最も衝撃的なシーンを持ってくること。

**推奨ハイライト案**
```
「ガチホしてる人、全員危ない。
リップル社は毎月3億枚を自分で売ってます。
そんな通貨を"買って放置"は──ナンセンスです。」
（ここでタイトルカット→本編導入へ）
```

**理由**：「ガチホ=危険」はリップルホルダー全員が「え？」と手を止める最強フック。本編③の核心部分だが、解決策（二刀流）はピー隠しにして好奇心を残す。
