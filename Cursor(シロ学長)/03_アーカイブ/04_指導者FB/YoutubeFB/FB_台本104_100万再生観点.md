# 指導者FB：台本104「ビットコインの知識ゼロから5年で1億円を作る無敵のロードマップ」
**観点：** 100万再生を取れるかという視点でのフィードバック
**対象台本：** 台本104（Q&Aエクスポート 2026/5/12）
**FB取得日：** 2026年5月13日

---

## 全体評価（指導者コメント）

台本の情報量・構成力は非常に高く、ロジカルな説得力もある。ただし、100万再生を狙うという観点では以下の課題がある：

1. **冒頭の離脱リスクが高い**（最初の15秒で「自分ごと」にできていない）
2. **中盤の専門用語・データ密度が高すぎる**（新規・ライト層がついてこれない）
3. **ピークエンドの法則が活かしきれていない**（1番盛り上がるべきポイントが分散している）
4. **CTA・セールスパートが長すぎる**（視聴維持率の急落を招く）

**最重要の3つの改善アクション：**
1. 冒頭を「貯金してる人、全員損してます」に変更 → 日本人の大半を「俺じゃん」にする
2. ③のセールスパートを3分以内に圧縮 → 視聴維持率の崩壊を防ぐ
3. サムネイルに「月3万円」「貯金=負け組」を入れる → 投資未経験の新規層を取り込む

## FBの全内容（10項目）

---

### 1. 冒頭（最初の15秒）— 最重要パート

**改善前：**
```
ビットコインなら老後2000万円問題を5年で解決できます。
・ビットコイン保有者は世界でたった7%
・アメリカ政府・企業が新規発行の2倍以上のペースで爆買い
```

**改善後：**
```
「貯金してる人、全員損してます。」

今あなたの銀行口座に眠っている100万円、5年後には実質80万円の価値しかなくなります。
一方で、5年前にビットコインを100万円分買った人は、今1,200万円になっています。

この差、12倍。
しかも今、ビットコインを持っている日本人はたった5%。
残りの95%の人は、この事実を知らないまま貯金し続けています。
```

**何がどう治ったか：**
- 「老後2000万円問題」→「貯金してる人、全員損してます」：既に語り尽くされたワードから、貯金派の全日本人が「俺じゃん」と感じるフックに変更
- 「ビットコイン保有者が7%」という他人事の数字→「あなたの100万円が80万円になる」という自分ごとの損失に変換（損失回避バイアスを刺激）
- 「12倍」という具体的な数字でベネフィットを即座に提示

**抽象ルール：**
> 冒頭は「自分の貯金・お金が実は減っている」という身近な損失から入る。抽象的な社会問題ではなく「あなたの100万円」「5年後には80万円」のように視聴者の財布で語る

### 2. 自己紹介パート

**改善前：**
```
株式・仮想通貨で年間10億円を運用するシロです。
2018年に300万円を溶かしました。
なので今日の話には自信があります。
```

**改善後：**
```
株式・仮想通貨で年間10億円を運用するシロです。
ただ、最初から上手くいったわけじゃありません。
2018年に300万円を一瞬で溶かして、半年間チャートを見るのが怖くなりました。
だからこそ、初心者が絶対にやってはいけないことも、今日全部話します。
```

**何がどう治ったか：**
- 「なので今日の話には自信があります」→「初心者がやってはいけないことも話す」：視聴者にとってのベネフィットが不明確だったものを、動画を最後まで見る動機に変換
- 「300万円を溶かした」→「一瞬で溶かして、半年間チャートを見るのが怖くなりました」：数字だけの失敗談から感情を伴うエピソードに変えることで共感度UP

**抽象ルール：**
> 自己紹介の失敗談は数字（300万円）だけでなく、その後の感情状態（「怖くなった」「眠れなかった」等）をセットにすることで視聴者との距離が縮まる。失敗談の後には必ず「だからこそ今日〇〇を話す」とベネフィットに繋げる

### 3. 3つの話題の提示部分（目次）

**改善前：**
```
①ビットコインの現状と今仕込むべき理由
②BTCが5年で1億円に値上がりする3つの根拠
③ビットコインで老後2000万円問題を解決するロードマップ

③が1番大事なので、絶対に最後まで見てください！
```

**改善後：**
```
今日話すのはこの3つ。

① なぜ今、世界中の政府と企業がビットコインを奪い合っているのか
② ビットコインが5年で1億円になると言い切れる3つの証拠
③ 手取り25万円の人が、月3万円の積立だけで老後2000万円を作る具体的なステップ

特に③は、今日この動画を見た人だけが得する話なので、最後まで見逃さないでください。
```

**何がどう治ったか：**
- 「現状と今仕込むべき理由」（抽象的）→「なぜ今、政府と企業が奪い合っているのか」（好奇心を刺激するワード）
- 「老後2000万円問題を解決するロードマップ」（漠然）→「手取り25万円の人が月3万円で〇〇を作る」（ターゲットの具体的なペルソナに落とす）
- 「絶対に最後まで見てください！」→「今日この動画を見た人だけが得する」（命令形からベネフィット訴求に変換）

**抽象ルール：**
> 目次の各項目にはターゲットが「自分のことだ」と感じる具体的な属性数字（手取り25万円・月3万円）を入れる。③の予告は「一番大事」より「今日この動画を見た人だけが得する」の方がベネフィットが明確

### 4. 中盤の市場データ解説（①のセクション）

**改善前：**
```
①：大口投資家が売り始めても価格が落ちない
直近、ビットコインが8万ドルを回復したことで、お金持ちの投資家たちが少しずつ売り始めています。
その金額は、なんと直近2週間の平均で1日当たり2800億円。

②：82,500ドル付近に買い注文が殺到
2つ目は、82,500ドル付近に約3000億円規模の買い注文が入っているからです。

③：99%のトレーダーが「下がる」に賭けているから
```

**改善後（大幅短縮）：**
```
じゃあ、今買って大丈夫なの？という話。
結論、プロのデータを3つだけ見せます。

1つ目。今、大口投資家が1日2,800億円分売ってるのに、価格が全く落ちてない。
なぜか？個人投資家が売らずに持ち続けてるのと、ブラックロックが即座に買い戻してるから。
つまり「売りたい人より買いたい人の方が圧倒的に多い」ということ。

2つ目。今の価格のすぐ上に、約3,000億円分の買い注文が待機してる。
ダムの水がギリギリまで溜まってる状態。決壊したら一気に上がる。

3つ目。プロのトレーダーの99%が「下がる」に賭けてる。
でも価格が上がった瞬間、全員が損切りで買い戻す。
つまり、上がり始めたら止まらなくなる構造になってる。

この3つが揃ってる今、仕込まない理由がありますか？
```

**何がどう治ったか：**
- 各データの丁寧な説明（「82,500ドル」「ショートスクイーズ」等の専門用語）→ 「要するにこういうこと」を3行で完結させる圧縮構成に変更
- 説明で終わっていたデータ解説→「仕込まない理由がありますか？」で視聴者に能動的に考えさせる問いかけで締める

**抽象ルール：**
> 市場データを解説するとき、100万再生狙いの動画では専門用語の詳細説明より「つまりどういうことか」の3行結論が優先。データ提示の締めには「○○な理由がありますか？」という視聴者への問いかけを置く

### 5. 半減期の解説

**改善前：**
```
根拠①：半減期サイクルが示す「上昇」
2012年：12ドルから1000ドルで約90倍
2016年：650ドルから20,000ドルで約31倍
（数字の羅列を先に提示）
```

**改善後：**
```
ビットコインには4年に1回、必ず起きるイベントがあります。それが半減期。

簡単に言うと、「新しく作られるビットコインの量が、ある日突然半分になる」という仕組み。

お米で例えますね。
毎月100万トン届いてたお米が、ある日から50万トンしか届かなくなる。
でも食べる人の数は変わらない。値段、上がりますよね？

これが過去4回起きて、4回とも値上がりしてます。例外ゼロ。

（チャート表示）
最初は90倍、次は31倍、その次は8倍、直近は4.7倍。

「倍率下がってない？」と思った方、鋭い。
でも1回あたりの利益額で見てください。
2016年は1BTCあたり300万円の利益。2024年は1,400万円の利益。
金額ベースでは5倍に増えてる。
```

**何がどう治ったか：**
- 数字の羅列を先に出す構成→「仕組みの説明（例え話）→チャート→数字」の順番に変更（理解の土台を先に作ってから数字を提示）
- タイトル「半減期サイクルが示す『上昇』」（抽象的）→「4年に1回、必ず起きるイベント」（具体性・必然性が伝わる）
- 「例外ゼロ」という断言ワードを追加し、信頼と好奇心を同時に刺激

**抽象ルール：**
> 専門的な仕組みの解説は「仕組みの例え話 → 実データ → 数字の解釈」の順番。数字を先出しすると理解できず離脱する。「例外ゼロ」「一度もない」等の断言ワードで信頼を作る

### 6. セールスパート（ロードマップ③）— 最大の改善ポイント

**問題点：**
現状の台本では③のセクションが実質的にセールス（メルマガ・コミュニティ誘導）になっており、全体尺の約40%を占めている。これは視聴維持率の大幅な低下を招く。

| 項目 | 改善前 | 改善後 |
|---|---|---|
| ロードマップ解説 | 長い（積立→トレード→元本作り4選→トレードで負ける理由→コミュニティ紹介） | 積立シミュレーション1本に絞り、セールスは最後2分以内に凝縮 |
| 投資元本を作る方法4選 | 物販・転職・節約・トレードの4つを紹介 | 丸ごとカット or 30秒に圧縮（「元本を増やす方法は色々ありますが、詳しくはメルマガで」） |
| トレードで負ける理由 | 自己流・自動売買ツールの2つを詳細解説 | 「独学で99%が負ける」の一言 + 「だから環境が大事」で即CTA |

**改善後の③セクション（大幅短縮版）：**
```
じゃあ具体的に何をすればいいのか？
やることはシンプルで、毎月一定額をビットコインに積み立てるだけ。

仮にビットコインが5年後に1億円になるとして、
月3万円の積立を今日から始めた場合、5年後の資産は約660万円。
月5万円なら約1,100万円。月10万円なら約2,200万円。

（表をテロップで表示）

「月3万円じゃ2,000万円届かないじゃん」って思いましたよね？
その通りです。だから元本を増やす工夫が必要になる。

ここが1番大事なんですが、長くなるのでこの話は次の動画で詳しくやります。
今日のところは「とにかく月3万円でいいから今日始める」これだけ覚えてください。

ちなみに、元本の増やし方やトレードの基礎を体系的に学びたい方は、
概要欄のメルマガに登録しておいてください。
今なら5つの特典を無料で受け取れます。
```

**何がどう治ったか：**
- 40%を占めていたセールス・コミュニティ紹介を大幅カット。「次の動画で詳しく」という予告に変換することでチャンネル登録と次回視聴の動機付けに転換
- トレードで負ける理由の詳細解説→「独学で99%が負ける」一言+「だから環境が大事」で即CTA（情報→行動の導線をシンプルに）
- セールスパートを2分以内に凝縮

**抽象ルール：**
> 100万再生を取る動画は「情報提供型」が主軸。セールス比率が高いとYouTubeアルゴリズムが「広告的な動画」と判断しインプレッションを絞る傾向がある。③セクションでトレードや環境の話を詳細展開するなら「次の動画で詳しく」と予告して次回視聴の動機に転換する

### 7. 前半CTAの挿入位置と内容

**改善前：**
```
ちなみに、この動画で話しきれない最新情報やトレードノウハウは、メルマガで配信してます！
今だけ期間限定で豪華5大特典も受け取れるので...
また、後半では、売り買いのタイミングが自動表示されるツールの導入方法もご案内するので...
```

**改善後：**
```
ちなみに、この動画の内容をPDF1枚にまとめた資料を概要欄で配ってます。
「ビットコインが1億円になる3つの根拠」を人に説明したい時にそのまま使えるので、
欲しい方は概要欄からどうぞ。
```

**何がどう治ったか：**
- 「話しきれない最新情報」「豪華5大特典」（盛りすぎて売り感が出る）→「PDF1枚にまとめた資料」（具体的で受け取るハードルが低い）に変更
- 「人に説明したい時に使える」という視聴者にとっての明確なベネフィットを追加
- 「ツールの導入方法もご案内」という段階的な早すぎるCTA→削除（本編に集中させる）

**抽象ルール：**
> 前半CTAは「豪華5大特典」等の装飾的表現ではなく、「PDF1枚」「人に説明する時に使える」のような具体的で受け取りハードルの低いものに絞る。「ツールの案内は後半で」という告知も前半では不要

### 8. まとめパート

**改善前：**
```
最後に今日の話を30秒でまとめます！
1つ目：今のビットコインは昔のスマホと同じく普及率7%...
2つ目：ビットコインが5年で1億円に成長する根拠...
3つ目：ビットコインで老後2000万円を解決するためにやるべきことは...
```

**改善後：**
```
最後に1つだけ。

2008年、iPhoneが日本に上陸した時、
「あんなの流行らない」「ガラケーで十分」と言った人がほとんどでした。

2025年の今、ビットコインに対して同じことを言っている人がいます。

歴史は繰り返す。
違うのは、今回はあなたが「早い側」に回れるということ。

月3万円でいい。今日始めるだけでいい。
5年後の自分に「あの時始めてくれてありがとう」と言われる選択をしてください。
```

**何がどう治ったか：**
- 箇条書きの復習形式→感情を動かすストーリー形式に変更（まとめパートで「もう新情報ないな」と感じると離脱されるため）
- ピークエンドの法則を活用：最後に感情を動かすシーンを置くことで動画全体の満足度が上がる
- 「月3万円でいい。今日始めるだけでいい。」という再現性・簡単さの強調で行動喚起を後押し

**抽象ルール：**
> まとめパートは箇条書きの復習をしない（「もう情報ない」と判断されて離脱される）。ピークエンドの法則に基づき、感情を動かすストーリー（時代対比・対比構造）で締めることで視聴満足度が最大化する

### 9. エンディングのコメント誘導

**改善前：**
```
「あなたは今、ビットコインを持ってますか？持ってないですか？」
持ってる人は今の枚数、持ってない人は「まだ」とコメント欄に書いてください♪
```

**改善後：**
```
最後に1つだけ質問させてください。
「もし5年後にビットコインが1億円になってたら、あなたは何に使いますか？」
コメント欄に書いてみてください。

ちなみに僕は、親を旅行に連れていきたいです。
```

**何がどう治ったか：**
- 「持ってる/持ってない」（二択で会話が広がらない）→「何に使うか？」（未来を想像させる問いに変換）：コメントが長文化しやすく、エンゲージメントが上がる
- 「ちなみに僕は〜」という人間味のあるコメントを追加することでファン化を促進
- 「まだ」という指定コメント→自由記述型に変更することで多様なコメントが集まりやすくなる

**抽象ルール：**
> エンディングのコメント促進は「持ってる/持ってない」等の二択より、「5年後に〇〇になったら何に使いますか？」のような未来想像型の問いの方がエンゲージメントが高い。演者自身の答え（「親を旅行に〜」）を添えることでファン化につながる

### 10. サムネイル・タイトルへのフィードバック

**サムネイル改善：**

| 要素 | 改善前 | 改善後 |
|---|---|---|
| 左上（フック） | 今さら聞けない | 「貯金してる人、全員負け組」or「95%の日本人が知らない」 |
| 右上（具体） | イーロンも大量保有 | 「100万円 → 1,200万円」（過去実績の数字） |
| 左下（ベネフィット） | 1億円 | 「月3万円で老後解決」 |
| 右下（好奇心） | — | 演者の驚いた表情 or「？」マーク |

**タイトル改善：**

改善前：
> 【今さら聞けない】ビットコインが5年で1億円になる理由

改善後：
> 貯金だけの人が5年後に後悔する理由｜ビットコイン完全解説
> or
> 月3万円で老後2000万円を作る方法｜ビットコイン積立の全て

**何がどう治ったか：**
- 「今さら聞けない」（既存の投資層には刺さるが投資未経験層には「自分には関係ない」と映る）→「貯金してる人、全員負け組」（貯金派の全日本人が「俺じゃん」と感じる）
- 「1億円」（大きすぎて現実感がない）→「月3万円で老後解決」（再現性・簡単さが伝わる）
- 「今さら聞けない」（SEOキーワードとして弱い）→「貯金だけの人」「月3万円」（ターゲット属性が明確でクリック率が高い）

**抽象ルール：**
