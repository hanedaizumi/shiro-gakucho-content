## 1. テクニカルチャンネルにおけるキャラ設定

### 立ち位置の根本的な違い

| | Web3チャンネル | テクニカルチャンネル |
|---|---|---|
| **役割** | 地獄を知る"兄貴分"・先輩 | 予測を当て続ける"分析の教祖" |
| **関係性** | 「その気持ち、分かるよ」と寄り添う | 「私についてきてください」と導く |
| **権威の源泉** | 地獄からの這い上がりストーリー | チャートを読む精度・予測の的中率 |
| **視聴者に与えるもの** | 共感・安心・背中を押す言葉 | 具体的な行動指針・エントリー根拠 |
| **禁止ワード** | 「楽して稼げる」「誰でも」 | 「絶対に当たります」「直感で」「なんとなく」 |

---

### テクニカルチャンネルにおける「キャラの核心」

> **「私は予測しているのではない。チャートの構造を読んでいるだけだ。」**

このチャンネルにおけるシロ学長は「天才」でも「ギャンブラー」でもない。
- 感情を完全に排除し
- チャートの構造・ルール・ロジックだけで動く
- 再現性のある手法を持ち
- 毎回「前回の予測が当たった」という実績を積み上げていく

**視聴者に植え付けたいイメージ：**
「この人の動画を見ていれば、自分も同じ目線でトレードできる。見ていない動画で損をしたくない。」

---

### プロフィール（テクニカルチャンネル用）

**【短縮版（冒頭で毎回使う）】**
```
株式・仮想通貨で年間10億円を運用するシロです。
```

**【標準版（権威性エピソードつき）】**
```
8年前に感情任せのトレードで1ヶ月で数千万円を溶かした私が、
今は月利1,000万円を安定して出せるようになりました。
理由は1つ。チャートは感情を持たない、という事実に気づいたからです。
株式・仮想通貨で年間10億円を運用するシロです。
```

**【実績強調版（信頼が必要な場面）】**
```
野村証券・仮想通貨取引所bitbankでの現場経験を持ち、
現在は株式・仮想通貨で年間10億円を運用しているシロです。
過去に数千万円を溶かした経験があるからこそ、
今日お伝えするラインの根拠には絶対の自信があります。
```

**バリエーションの使い分けルール：**
- 動画1〜3本目：標準版（失敗 → 成功のストーリーで共感させる）
- 4本目以降：短縮版を基本としつつ、重要回は実績強調版を使う
- エピソードの中身は毎回少しずつ変えて使い回し感を消す

## 2. 「教祖化」を加速させる権威の積み上げ方

教祖としての信頼は1本の動画では作れない。**毎回の動画で少しずつ積み上げる**もの。

### 積み上げのサイクル（毎動画で繰り返す）

```
①【前回の予測を検証】（動画冒頭）
   「前回、〇〇ドルを割れたらショートと言いました。見事に割れ、○○ドル幅取れた方おめでとうございます。」
        ↓
②【今回の新しい根拠を説明】（本編）
   「今週注目すべきは〇〇。なぜなら〜」（具体的なラインとロジック）
        ↓
③【行動プランを渡す】（アクションプラン）
   「今週やることは1つだけ。〇〇ドルにアラームをセット。」
        ↓
④【次回予告で引き留める】（エンディング）
   「次回は〇〇ドルに達した場合の対応を解説します。見逃すと損します。」
```

このサイクルを繰り返すことで「この人の動画を見ていれば、先にリスクを回避できる」という信頼が生まれる。

---

### 教祖化フレーズの使い方

| 場面 | フレーズ | 狙い |
|------|---------|------|
| 冒頭 | 「前回の動画で〇〇ドルを割れると言いました。見事に的中しました。」 | 実績の蓄積 |
| 冒頭 | 「前回の動画を見ていた方は、この〇〇ドル幅のチャンスを取れたはずです。」 | 「見ない損失」の演出 |
| 本編 | 「チャートは嘘をつかない。あるのは構造だけです。」 | システムへの信頼 |
| 本編 | 「機関投資家がどう動いているか、チャートに全部書いてある。」 | 排他的情報の提示 |
| 本編 | 「私は予測しているのではなく、構造を読んでいます。」 | 直感ではなくロジック |
| 根拠説明 | 「条件が揃ったら動く。揃ってなければ待つ。それだけです。」 | シンプルで再現性のある印象 |
| まとめ | 「今日やることは1つだけ。〇〇ドルにラインを引く。」 | 具体的で実行しやすい |
| エンディング | 「次回は〇〇について解説します。今週のトレードに直結するので、必ず見てください。」 | 次の動画への期待感 |

## 4. テクニカルチャンネル専用エピソード（使用優先度順）

テクニカルチャンネルでは「感情 vs ロジック」の文脈でエピソードを使う。
全エピソードの詳細は `persona_story_phrases_20260602.md` を参照。

### 優先度A（必ず使えるもの）

**感情トレードで数千万円を溶かした経験（8年前）**
```
8年前に感情任せのトレードで、1ヶ月で数千万円を溶かしたこともあります。
でも今は、チャートの形とルールに従って機械的に取引するようになってから、
安定して月利1,000万円以上を出せるようになりました。
必要なのはセンスではなく、正しいロジックと環境の2つ。これだけは絶対に覚えておいてください。
```
→ **使用シーン：** エモい話・エンディング（毎回使えるが、後半のエピソードの内容は少しずつ変える）

**野村証券の「伝説のトレーダー」上司の言葉**
```
野村証券時代、「なぜあなたは勝ち続けられるんですか？」と聞いた上司にこう言われました。
「俺は天才じゃない。感情を殺す方法を知っているだけだ。トレードで負ける理由の99%は恐怖と欲望の感情。
これをルールで縛れば、誰でも勝てる」
この言葉が今の私の全てのベースになっています。
```
→ **使用シーン：** 感情トレードの危険性・ルールの重要性を説明するパート

### 優先度B（テーマに合わせて使うもの）

**ビギナーズラックでボーナス全額突っ込んだ失敗**
→ **使用シーン：** 「自分は分かっている」という過信への警告・ダニング＝クルーガー効果の説明

**3年間の塩漬け地獄（500万円→200万円）**
→ **使用シーン：** ガチホだけではダメな理由・短期トレードとの両輪戦略の説明

**マニラのVIPカジノでトレードとギャンブルの違いを確信**
→ **使用シーン：** 損切りの重要性・カジノとトレードの違いを説明するパート

### 優先度C（特定のテーマ専用）

**2024年8月にBTC800万円で1枚仕込み、4ヶ月で1600万円に**
→ **使用シーン：** 現物の積み立て・タイミングの話・「準備と実行」の成果事例

**東大卒エリート同僚が全財産消滅**
→ **使用シーン：** 「頭の良さ=投資の強さ」ではないことの証明

## 5. テクニカルチャンネルで絶対にやらないこと（NG）

| NG行為 | 理由 |
|--------|------|
| 「絶対上がります」「確実に下落です」 | 外れた瞬間に全ての信頼が消える |
| 「今すぐ全力で入れてください！」 | リスク管理と矛盾。教祖が崩れる |
| 「私のロジックは100%です」 | 嘘になる。謙虚さが信頼を保つ |
| エピソードなしの自己紹介（「シロです」だけ） | 権威性が伝わらない。必ずエピソードを1文添える |
| 根拠なしの方向感（「感覚では下だと思います」） | 感情・直感トレードと同じ。論理がない |
| 概念の説明をスキップ（「ダウ理論は知ってますよね？」） | 中級者すら切り捨てる。10秒で定義を必ず入れる |
