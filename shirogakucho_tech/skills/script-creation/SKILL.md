# YouTube台本作成スキル（テクニカルチャンネル専用）

## 事前読み込み（必須）

作業開始前に以下を必ず読み込む：

- `00_チャンネル設計前提.md`（チャンネルコンセプト・権威性・NG表現）
- `03_YouTube台本/インプット/persona_technical.md`（テクニカルチャンネル専用キャラ定義・フレーズ・エピソード）
- `03_YouTube台本/インプット/` 内の最新台本（直近1〜2本 — 語り口・価格帯の参照）
- `03_YouTube台本/インプット/参考台本.md`（競合ベンチマーク）

---

## 台本全体の最重要目標

> **視聴者に「ちゃんと理解できた。自分もやってみたい。だからDiscordに入ろう」と思わせること。**
> 情報を詰め込むのではなく、1本を通して「分かる体験」を積み重ね、最終的にDiscord参加という自然な行動に誘導する。

---

## 台本の基本構成（5本目以降 固定テンプレート）

以下の順序を必ず守ること。変更は厳禁。全体で10分（約3,200〜3,500文字）に収める。

```
① 導入（冒頭フック）
② 前半CTA（Discord誘導）
③ 目次（今日話す3点）
④ BTCの現在地（日足分析）
⑤ 今週の重要ポイント（中級者向けの概念を1つ、しっかり解説）
⑥ コメント誘導（インタラクション）
⑦ 上昇・下落 両シナリオのアクションプラン
⑧ 中盤CTA（Discord + Vantageを軽く）
⑨ まとめ（30秒）
⑩ エモい話（自己開示 × 視聴者への激励）
⑪ 後半CTA（Discord + 次回予告）
⑫ 締めの挨拶
```

**スタイル原則：**
- 不要な問いかけ・つなぎ言葉は削除する（「ここで皆さんに質問ですが…」などの引き伸ばし禁止）
- チャートをベースに淡々と解説するスタイルを徹底
- 4時間足分析は⑦の両シナリオ内で価格を示す形で組み込む（独立したセクションは不要）

---

## 各セクションの書き方ルール

### ① 導入（冒頭フック）

**目標：** 最初の10秒で「見続ける理由」を与える

**構成（順番を守ること）：**
1. **衝撃の1文**（今のBTCで起きていることを断言）
   - 例：「今のビットコイン、買い場だと思ってエントリーすると機関投資家のカモにされます。」
   - 「可能性」「かもしれない」は絶対禁止
2. **前回予測の的中報告**（②以降の動画で必ず入れる）
   - 「前回の動画で○○ドルのラインを割れると言いました。見事に的中し、ショートで〇〇ドル幅の利益が取れました。」
   - 利益を取れた視聴者への「おめでとうございます！」
   - 見ていなかった人への「○○ドル幅のチャンスを逃した形です」で悔しさを演出
3. **権威性の提示（エピソードつき）** ← 【施策①】
   - ❌現状（NG）：「株式・仮想通貨で年間10億円を運用するシロです。」だけ
   - ✅今後（OK）：以下のフォーマットで権威を「体験談」とセットにする
   ```
   8年前に感情任せのトレードで、1ヶ月で数千万円溶かした私が、
   今は月利1,000万円を安定して出せるようになった。
   理由は1つ。「チャートは感情を持たない」という事実に気づいたからです。
   株式・仮想通貨で年間10億円を運用するシロです。
   ```
   - エピソードは毎回少しずつ変えて使い回さない（`03_YouTube台本/インプット/` の直近台本と被らないこと）
4. **今日の動画で得られるベネフィット（1文）**
   - 「最後まで見れば、上昇・下落どちらに転んでも1,000ドル幅の利益が取れるようになります。」

---

### ② 前半CTA（Discord誘導）

**目標：** Discord参加を自然な流れで促す。短く、本編の邪魔をしない。

```
また、既に○○名近いメンバーが在籍するシロ学長のDiscordコミュニティでは、
今日の狙い目とリアルタイムの市況解説を毎日無料で配信しています！

私が実際にトレードで使っているインジケーターもメンバー限定で無料配布中。
概要欄のリンクからご参加ください！
```

- Vantage紹介はここには含めない（中盤CTAに回す）← 【施策②】

---

### ③ 目次（今日話す3点）

```
今日は3つお話しします。
①BTCの現在地
②[今週の重要ポイント名]
③上昇・下落シナリオとアクションプラン

特に3つ目はトレードに直結してくるので最重要です！
```

**バリエーション例（「最重要です」の言い換え）：**
- 「3つ目だけはメモを取りながら見てください。」
- 「3つ目は日本語で解説している動画がほぼないので、最後まで集中して見てください。」
- 「3つ目を知っているかどうかで、来週の結果が全然変わります。」

---

### ④ BTCの現在地（日足分析）

**目標：** 状況を明確に整理し「基本目線」を確立する

**分析の3観点（必ずこの順序で解説すること）：**

```
【観点1】指標（インジケーター）
- 移動平均線の並び（パーフェクトオーダー上向き/下向き）
- RSI・MACDなど補助指標の状態

【観点2】ローソク足
- ダウ理論の活用（高値・安値の切り上げ/切り下がりで判定）
- 直近のローソク足の特徴（上ヒゲ陰線が多い → 売り圧強い、など）
- 直近の出来高との組み合わせ

【観点3】ライン（水平線・斜めライン）
- 意識されている水平サポート・レジスタンスライン（具体的な価格を必ず示す）
- トレンドライン・チャネルライン（斜めライン）
- 現在価格が上か下か、どのライン間に挟まれているか
```

**ポジションの原則：**
- 事前指示（ユーザーが伝えた相場観）をベースにしつつ、**必ず逆方向の可能性も1文添える**
- 「基本目線は〇〇だが、〇〇ドルを実体で上抜けた場合はその限りではない」という中立スタンスを維持する
- 断言はせず「〜と判断できます」「〜の可能性が高い」で統一

**その他の書き方ルール：**
- 数字は必ず具体的に（「重要なライン」ではなく「7万7,500ドル」）
- 専門用語は最初の1回だけ簡単に説明する（「ダウ理論とは〜のことです」を10秒で）
- 30秒でまとめられる「現在地1文」で締める

---

### ⑤ 今週の重要ポイント（中級者向けの概念を1つ）← 最重要セクション

**目標：** 「この人、俺の知らないことを知っている！」という体験を1回作る

**選ぶ概念の基準：**
- 中級者なら名前を聞いたことがある、でも「ちゃんとは分かってない」レベルのもの
- ✅適切な例：ダウ理論・フィボナッチ・ファンディングレート・未決済建玉・ボリュームスパイク・エリオット波動・オーダーブロック・上昇ウェッジ・逆三尊
- ❌NG：完全な初心者向けすぎる（ロウソク足とは？）or 上級すぎて全員置いてけぼり

**解説の流れ（必ず守ること）：**
1. 簡単に定義：10秒で「これは〜のことです」(そもそも何なのか？を説明)
2. 今のBTCチャートでの具体的な見方・使い方
3. 「これが分かると何が良いのか？」を説明
4. 実際のエントリー判断への繋ぎ：「だから今週は〜に注目します」

**使用済み概念トラッキング（被り防止）：**

| 台本 | 使用した概念 |
|------|------------|
| 台本① | 上昇ウェッジ・ダブルトップ・サポートライン割れ |
| 台本② | ファンディングレート＋未決済建玉 |
| 台本③ | フィボナッチ0.618 |
| 台本④ | 上ヒゲ連発の抵抗ライン・出来高でダマシを見抜く |

→ **次回（5本目）候補：ダウ理論の厳密定義・エリオット波動・オーダーブロック・逆三尊の正しい見方**

---

### ⑥ コメント誘導（インタラクション） ← 【施策③】

**目標：** アルゴリズム評価を上げるコメントを生む

**タイミング：** ⑤の重要ポイント解説の直後（まだ視聴者が画面を見ているタイミング）

**フォーマット例（毎回変える）：**
```
ちょっとここで質問です。
今の相場、あなたは上昇派ですか？下落派ですか？
コメントで「上」か「下」か教えてください。全部読んでます。
```
```
今日の重要ポイント、[概念名]を知ってた方はコメントで「知ってた」と教えてください。
知らなかった方は「初めて知った」でOKです。
```

**ルール：**
- 答えやすい二択形式にする
- 「全部読んでます」を必ず入れる（参加障壁を下げる）

---

### ⑦ 上昇・下落 両シナリオのアクションプラン

**目標：** どちらに転んでも批判されない構成 + 具体的な行動計画

**フォーマット（必ず両シナリオ提示）：**

```
【下落シナリオ（基本目線）】
トリガー：〇〇ドルを4時間足実体で割る
エントリー：ショート
損切り：〇〇ドル付近
利確第1目標：〇〇ドル
利確第2目標：〇〇ドル
注意：先回りせず実体確定を待つ

【上昇シナリオ（逆転目線）】
トリガー：〇〇ドルを4時間足実体で上抜け
エントリー：一旦戻りを待ってからロング
損切り：〇〇ドル割れ
利確第1目標：〇〇ドル
```

**共通ルール：**
- ポジションサイズは口座の2%まで（必ず1文入れる）
- 「サインが揃っても外れる時は外れる。だから機械的に2%」

---

### ⑧ 中盤CTA（Discord + Vantageを軽く） ← 【施策②・⑦の直後】

**目標：** 「Discord → その中でVantageの手順を確認」という導線を作る

```
こういったリアルタイムの狙い目や毎日の市況は、
シロ学長のDiscordコミュニティで無料配信しています。
より早く正確な情報でトレードしたい方は概要欄のリンクから参加をお願いします！

また、実際にトレードするには取引所が必要ですが、
僕が使っているVantageは手数料が安くておすすめです。
Discord内に詳しい登録手順を載せているので、まずはDiscordに入ってみてください。
```

**ポイント：**
- Vantageの紹介はこの中盤1回のみ（末尾からは削除する）
- 「Discordに入れば全部分かる」という導線にする

---

### ⑨ まとめ（30秒）

**目標：** 今日話した内容を圧縮して「持って帰れる知識」にする

**フォーマット：**
```
今日の内容を30秒でまとめます。

今のビットコインは[基本目線]。
[重要ライン]が意識すべきライン。

[今週の重要ポイント名]で見ると、[今のBTCのシグナル]が出ています。

[下落シナリオ]：[トリガー]で[エントリー方向]
[上昇シナリオ]：[トリガー]で[エントリー方向]

今日やることは1つだけ。
[具体的な行動（例：72,600ドルにアラームをセットする）]
```

---

### ⑩ エモい話（自己開示 × 視聴者への激励）

**目標：** 「この人を応援したい・信じたい」という感情を作る

**構成：**
1. **シロ学長の過去のエピソード（1〜2文）**
   - 「8年前に感情任せのトレードで1ヶ月で数千万円を溶かしたこともあります。」
   - 毎回同じにしない。エピソードはローテーション（下記参照）
2. **転換（1文）**
   - 「でも今は、チャートの形とルールに従って機械的に取引するようになってから、安定して月利1,000万円以上を出せるようになりました。」
3. **視聴者へのメッセージ（1〜2文）**
   - 「必要なのはセンスではなく、正しいロジックと環境の２つ。絶対あなたにもできます。」

**エピソードローテーション（同じ話を連続で使わない）：**

| 使用タイミング | エピソード |
|------------|---------|
| 台本①④ | 数千万円の損失 → 今は月利1000万 |
| 台本② | 500万円の損失 → 感情をコントロールできるようになった |
| 5本目候補 | 仕事中トイレでチャートを見ていた頃 → 今は自由な時間でトレード |
| 6本目候補 | ビギナーズラックでボーナス全額突っ込んだ失敗 |
| 7本目候補 | 野村証券の上司から「感情を殺す方法」を学んだ話 |

---

### ⑪ 後半CTA（Discord + 次回予告） ← 【施策④】

**目標：** Discord参加の最終促進 + 次回の連続視聴を生む

**Discord部分：**
```
シロ学長のDiscordコミュニティでは
・日々の狙い目
・リアルタイム市況
・相場分析ツール
これらが全て無料で手に入ります。

月額費用など一切かかりませんので、
概要欄のリンクからご参加をお願いします！
```

**次回予告部分（必ず入れる）：** ← 【施策④】
```
次回は、[次の動画のテーマや注目ポイント]についてお話しします。
[今週/来週]のBTCが[〇〇ドル]に届いたら、その後どう動くべきか。
それだけに絞って解説しますので、チャンネル登録してお待ちください。
```

**ルール：**
- 「次回は」で始める
- 具体的な価格帯や条件に言及する（「何かを話す」ではなく「〇〇ドルに届いたら」など）
- チャンネル登録の促進をここに自然に含める

## 台本作成の4施策チェックリスト

台本を出力する前に以下を必ず確認すること。

| 施策 | チェック内容 | OK/NG |
|------|-----------|-------|
| **①権威性** | 冒頭の自己紹介にエピソードつきの権威性アピールが入っているか | |
| **①権威性** | エピソードが直近2〜3本と被っていないか | |
| **②Vantage誘導** | 中盤CTAに1回だけVantageの紹介があるか | |
| **②Vantage誘導** | 末尾にVantageの単独誘導がないか（Discord内での導線になっているか） | |
| **③コメント誘導** | まとめの前にYes/No型のコメント誘導があるか | |
| **③コメント誘導** | 「全部読んでます」が入っているか | |
| **④次回予告** | 後半CTAに「次回は〜」の予告が入っているか | |
| **④次回予告** | 具体的な価格帯・条件に言及しているか | |
