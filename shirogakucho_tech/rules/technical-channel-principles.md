# シロ学長テクニカル｜常時適用ルール

## チャンネルの本質

> **シロ学長テクニカルは「トレードを教えるチャンネル」ではなく、「シロ学長の信者（ファン）を作るビジネス」。**
> 正しい情報を届けることは前提。その上で「この人の話をずっと聞きたい」と思わせる設計が必要。
> イメージはスピリチュアル系チャンネル。情報の質と深さが信者を作る。

## 常時チェック：3つの軸

コンテンツを生成・評価する際、常に以下の3軸を意識すること。

### ① わかりやすさ
- ロジックの流れが「前提 → 根拠 → 結論」になっているか
- 専門用語の直後に日常例えが入っているか
- 数字・条件が具体的か（「下がったら」→「7万5,500ドルを実体で割ったら」）

### ② 権威性
- 過去の予測との整合性が示されているか（「前回の動画で言った〇〇が的中」）
- 根拠が2段以上重なっているか
- 自己紹介でエピソードつきの権威性アピールが入っているか（`00_チャンネル設計前提.md`参照）

### ③ ファン化（情報で行うこと）
- 「この人俺の知らないこと知ってる！」と感じる新しい視点が1箇所以上あるか
- その気づきは情報・データ・ロジックで届けられているか（感情煽りではない）
- 必要のない箇所（挨拶・CTA）でファン化要素を差し込んでいないか

## 5本目以降の台本で必ず実施する4施策

### 施策① 権威性アピールの強化

**変更前（NG）：**
```
株式・仮想通貨で年間10億円を運用するシロです。
```

**変更後（必須）：**
```
8年前に感情任せのトレードで、1ヶ月で数千万円溶かした私が、
今は月利1,000万円を安定して出せるようになった。
理由は1つ。「チャートは感情を持たない」という事実に気づいたからです。
株式・仮想通貨で年間10億円を運用するシロです。
```

→ エピソードの内容は毎回変える。直近2本と被らないようにローテーションする。

---

### 施策② Vantage誘導を中盤1回のみに統一

**変更前（NG）：**
末尾でDiscordと合わせてVantageをガッツリ紹介

**変更後（必須）：**
1. 中盤CTAでDiscordを紹介した直後に一文でVantageに触れる
2. 「登録手順はDiscordの中にある」という導線にする
3. 末尾にVantageの単独訴求を入れない

→ 目的：視聴者を「まずDiscordに入る」「そこでVantageの手順も分かる」というシンプルな導線に乗せる。

---

### 施策③ 中盤でコメント誘導

**変更前（NG）：**
エンディングに「質問とかあればコメントでお願いします」の1行のみ

**変更後（必須）：**
まとめの直前に Yes/No・二択型のコメント誘導を入れる

```
例：「今日の相場、上昇派ですか？下落派ですか？コメントで教えてください。全部読んでます。」
```

→ 目的：視聴者がまだ画面を見ているタイミングでコメントを引き出す。アルゴリズム評価を上げる。

---

### 施策④ 次回予告で連続視聴率を上げる

**変更前（NG）：**
「今日もありがとうございました！」で終わる

**変更後（必須）：**
締めの挨拶の直前に次回予告を入れる

```
例：「次回は、〇〇ドルに届いた場合の対応と、その後のシナリオをお話しします。チャンネル登録してお待ちください。」
```

→ 目的：視聴者に「次の動画も見よう」という動機を持たせてチャンネル登録を促す。

## 絶対NG（常時）

- 「誰でも」「簡単に」「スマホで5分」などの煽り文言
- 根拠なき断言（「絶対上がります」「今が最後のチャンス」）
- 感情煽りによるファン化（「あなたは上位1%」「他の発信者は嘘をついている」）
- `00_チャンネル設計前提.md`にない実績・エピソードの創作
- ファン化要素を台本全体の30%超にする（情報の薄い動画になる）
- 「可能性があります」「かもしれません」→「〜と判断できます」「〜が高い」に統一

---

## 台本粒度の基準（参考動画スタイル）

以下2タイプを組み合わせること。

- **村上スタイル**：長期予測的中 × 自身の感情的体験 → 信頼と共感のセット
- **トレーダーRedスタイル**：直近予測の的中確認 → 条件付きシナリオ提示 → 具体的エントリーポイント
